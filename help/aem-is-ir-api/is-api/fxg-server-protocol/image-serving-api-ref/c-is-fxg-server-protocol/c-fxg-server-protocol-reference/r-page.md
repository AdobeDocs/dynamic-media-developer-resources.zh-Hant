# page{#page}

擷取頁面。 在多頁FXG中擷取特定頁面。

`page= *`val`*`

  <td class="stentry"> <p><span class="codeph"> <span class="varname"> val</span></span> </p> </td>

 </tr>

## 預設 {#section-3fd7fcc525b947c7a95457e50414ac9e}

如果未指定`page`，則會傳回第一頁以用於點陣輸出，以及所有頁面以用於PDF輸出。
