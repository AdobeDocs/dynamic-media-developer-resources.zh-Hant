# markOverfongTextFrames{#markoverflowingtextframes}

顯示帶加號的溢流文本框架。 當文字超出在文字框內為其分配的空間 (或者，如果是串連的文字，則為在最後一個文字框中分配的空間) 時，會顯示文字溢出指示器。該指示器是一個內含加號的紅色方塊。

<table id="simpletable_F17FD29EB52043BF9000923ED5195A26"> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph"> &amp;markOverfongTextFrames</span> </p> </td> 
  <td class="stentry"> <p>0|1 </p></td> 
 </tr> 
</table>

設定修飾符 `markOverflowingTextFrames=1` 通過URL調用標籤所有文本框，其中文本用加號疊加。 此外，在Dynamic Media Classic預覽器中，文本溢流指示器設定為&quot; `TRUE`」。

預設為 0。
