# printerMark{#printermark}

顯示打印機標籤。 指定如何顯示打印機標籤。

` printerMark= *`修剪標`*, *`記出`*, *`血標籤注`*, *`冊標籤顏`*, *`色標籤信`*, *``*, *`息樣式線`*, *`加權圖嵌入`*`

可以關閉或開啟不同的標籤。 還可以控制打印機標籤的樣式。
