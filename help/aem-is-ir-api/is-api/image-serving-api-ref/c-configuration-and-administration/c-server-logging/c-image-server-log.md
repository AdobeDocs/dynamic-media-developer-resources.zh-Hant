# 映像伺服器日誌{#image-server-log}

映像伺服器將記錄寫入此日誌檔案，以記錄警告和錯誤，並保留啟動和關閉事件的記錄。

此外，映像伺服器日誌可配置為跟蹤大量附加詳細資訊（如映像檔案訪問），以幫助調試伺服器問題。

映像伺服器日誌配置在[!DNL ImageServerRegistry.xml]中。
