# 影像伺服器記錄{#image-server-log}

影像伺服器會將記錄寫入此記錄檔，以記錄警告和錯誤，並記錄啟動和關閉事件。

此外，影像伺服器記錄檔可設定為追蹤大量其他詳細資訊（例如影像檔案存取），以協助偵錯伺服器問題。

影像伺服器記錄檔設定於 [!DNL ImageServerRegistry.xml].
