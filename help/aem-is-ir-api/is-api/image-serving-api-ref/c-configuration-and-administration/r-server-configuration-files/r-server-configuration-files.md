# 伺服器配置檔案{#server-configuration-files}

所有配置檔案都位於install_folder/conf中，大多數文本編輯器都可編輯。 伺服器可能需要重新啟動才能使更改生效。

>[!NOTE]
>
>大多數伺服器配置檔案包含本文檔中未說明的其他屬性和值。 這些屬性僅供內部伺服器使用，除非Scene7技術支援特別指示，否則不得修改。
