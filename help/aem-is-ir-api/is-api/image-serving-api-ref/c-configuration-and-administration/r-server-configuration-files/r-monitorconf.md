# monitor.conf{#monitor-conf}

包含監視／警報系統的設定。

此檔案是JAVA屬性檔案。 必須注意遵守適當的公約；否則平台伺服器可能無法啟動。 請注意，在Windows檔案路徑中，必須使用雙反斜線&#39;\\&#39;或單正斜線&#39;/&#39;，而非反斜線&#39;\&#39;，因為反斜線是此類檔案中的轉義字元。

此檔案的變更會在儲存檔案後不久生效。
