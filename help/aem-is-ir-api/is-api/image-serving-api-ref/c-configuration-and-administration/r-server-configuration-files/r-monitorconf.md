# monitor.conf{#monitor-conf}

包含監視/警示系統的設定。

此檔案是JAVA屬性檔案。 請務必遵循適當的慣例；否則[!DNL Platform Server]可能無法啟動。 請特別注意，在Windows檔案路徑中，必須使用雙反斜線&#39;\\&#39;或單正斜線&#39;/&#39;，而非反斜線&#39;\&#39;，因為反斜線在此型別的檔案中是做為逸出字元。

對此檔案的變更會在檔案儲存後不久生效。
