# PlatformServer.conf{#platformserver-conf}

包含 [!DNL Platform Server] 設定。

此檔案是JAVA屬性檔案。 必須注意遵循適當的公約；否則， [!DNL Platform Server] 可能無法啟動。 在Windows檔案路徑中，使用雙反斜線「\\」或單正斜線「/」，而非反斜線「\」。 反斜線在此類型的檔案中用作逸出字元。

此檔案的更改在保存檔案後生效。

只有下列設定可在 [!DNL PlatformServer.conf]. 如果特定設定不存在，則可在檔案中的任何位置新增該設定。 每個設定只能有一個例項。
