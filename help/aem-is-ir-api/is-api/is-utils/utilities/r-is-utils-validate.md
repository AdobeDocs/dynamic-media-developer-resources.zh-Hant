# 驗證{#validate}

映像驗證實用程式。 此命令行實用程式會驗證映像檔案，以確保它們有效，並且通過映像服務可以無困難地讀取。

所有非PTIFF影像檔案必須先通過驗證，才能將檔案提供給「影像伺服」做為來源影像。 PTIFF影像應在可能不可靠的復製作業後進行驗證。

` validate *`檔`* [ *``*] [ *`案類型來源檔案`* [ … ]]`

## 傳回{#section-67a7cf7c53144fbb8f24b818f4a10901}

0（如果成功）。 如果發生錯誤，則會傳回非零值，並將錯誤詳細資料傳送至`stderr`。
