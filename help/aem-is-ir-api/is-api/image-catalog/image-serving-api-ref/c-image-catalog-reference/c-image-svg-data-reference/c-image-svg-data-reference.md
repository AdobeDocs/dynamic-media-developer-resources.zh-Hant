# 影像/SVG資料參考{#image-svg-data-reference}

儘管影像和SVG資料非常相似，但它們仍保留在影像目錄中的單獨表中。

如果需要影像資料，則必須在中指定列出影像的影像資料檔案的相對路徑或絕對路徑 `attribute::CatalogFile`。 如果需要SVG資料，必須在中指定SVG資料檔案的相對或絕對路徑 `attribute::SvgCatalogFile`。
