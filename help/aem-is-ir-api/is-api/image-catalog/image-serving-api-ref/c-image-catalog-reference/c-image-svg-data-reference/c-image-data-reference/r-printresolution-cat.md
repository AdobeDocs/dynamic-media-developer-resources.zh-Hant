# 打印解析度{#printresolution}

打印解析度。 打印全尺寸影像的解析度。

此值嵌入到回復影像標題中，除非用 `printRes=`。

## 屬性 {#section-de3c1f73da7b43208beeec841c1778c1}

整數，大於0。 以點/英吋表示。 選擇性.

## 預設 {#section-0cac992554ec4247ab05f70d9840a045}

`attribute::PrintResolution` 如果欄位不存在，則使用。
