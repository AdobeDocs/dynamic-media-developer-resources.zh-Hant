# 打印解析度{#printresolution}

打印解析度。 打印全尺寸影像的解析度。

此值嵌入在回復影像標題中，除非以`printRes=`覆蓋。

## 屬性 {#section-de3c1f73da7b43208beeec841c1778c1}

大於0的整數。 以每英吋點表示。 選填。

## 預設 {#section-0cac992554ec4247ab05f70d9840a045}

`attribute::PrintResolution` 如果欄位不存在，則值為0或欄位為空，則會使用。
