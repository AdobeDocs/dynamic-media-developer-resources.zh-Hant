# AssetType{#assettype}

資產類型. 用於識別發佈至目錄的集類型：:ImageSet。

資產類型會決定要針對請求產生何種回應 `req=set` 類型。 如果未指定值，自動偵測規則會決定回 `req=set` 應類型。

## 屬性 {#properties}

下列集的列舉值：

MEDIASET

旋轉集

SPINSET2D

VIDEOSET

影像集

ECATALOG

## 預設 {#section-b94ce61423194729918456103ef3f72c}

無。
