# 資產類型{#assettype}

資產類型. 用於標識發佈到目錄：:ImageSet的集的類型。

資產類型確定要為生成的響應類型 `req=set` 請求。 如果未指定值，則自動檢測規則將確定 `req=set` 響應類型。

## 屬性 {#properties}

以下集中的枚舉值：

MEDIASET

旋轉集

SPINSET2D

視頻集

映像集

對話

## 預設 {#section-b94ce61423194729918456103ef3f72c}

無。
