# 資產型別{#assettype}

資產型別。 用於識別發佈至catalog：：ImageSet的集型別。

資產型別會決定要為`req=set`個要求產生哪種回應型別。 如果未指定值，則自動偵測規則會決定`req=set`回應型別。

## 屬性 {#properties}

來自下列集合的列舉值：

MEDIASET

迴轉集

SPINSET2D

VIDEOSET

影像集

ECATALOG

## 預設 {#section-b94ce61423194729918456103ef3f72c}

無。
