# AssetType{#assettype}

資產類型. 用於識別發佈至目錄：:ImageSet的集類型。

資產類型決定要為`req=set`請求產生的回應類型。 如果未指定值，自動檢測規則將確定`req=set`響應類型。

## 屬性 {#properties}

下列集的列舉值：

MEDIASET

SPINSET

SPINSET2D

VIDEOSET

影像集

ECATALOG

## 預設 {#section-b94ce61423194729918456103ef3f72c}

無。
