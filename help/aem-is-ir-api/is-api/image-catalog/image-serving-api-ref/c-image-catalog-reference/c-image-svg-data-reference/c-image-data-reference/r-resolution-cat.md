# 解析度{#resolution}

物件解析度。 完整大小影像的物件解析度。

用於使用`res=`命令以解析度為基礎的縮放。

## 屬性 {#section-861a86da50364e2181d62778338359ea}

大於0的實數。 通常以每英吋的畫素表示，但也可能以其他單位表示，例如每米的畫素。 選擇性.

## 預設 {#section-5b3a83d6410b43efa1fa3add396196f9}

如果欄位不存在、值為0或欄位為空，則會使用`attribute::Resolution`。
