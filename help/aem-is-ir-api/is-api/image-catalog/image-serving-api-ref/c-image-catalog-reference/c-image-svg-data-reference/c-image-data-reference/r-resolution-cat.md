# 解析度{#resolution}

對象解析。 全尺寸影像的對象解析度。

用於使用 `res=` 的子菜單。

## 屬性 {#section-861a86da50364e2181d62778338359ea}

實數，大於0。 通常以像素/英吋表示，但也可以以其他單位表示，如每米像素。 選擇性.

## 預設 {#section-5b3a83d6410b43efa1fa3add396196f9}

`attribute::Resolution` 如果欄位不存在，如果值為0，或者欄位為空，則使用。
