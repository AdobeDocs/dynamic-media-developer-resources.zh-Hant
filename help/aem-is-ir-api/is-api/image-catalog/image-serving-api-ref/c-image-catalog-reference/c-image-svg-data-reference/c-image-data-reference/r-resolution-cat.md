# 解析度{#resolution}

對象解析度。 全尺寸影像的對象解析度。

使用`res=`命令進行基於解析度的縮放。

## 屬性 {#section-861a86da50364e2181d62778338359ea}

實數，大於0。 通常以每英吋像素表示，但也可以以其他單位表示，例如每米像素。 選填。

## 預設 {#section-5b3a83d6410b43efa1fa3add396196f9}

`attribute::Resolution` 如果欄位不存在、值為0，或欄位為空，則會使用。
