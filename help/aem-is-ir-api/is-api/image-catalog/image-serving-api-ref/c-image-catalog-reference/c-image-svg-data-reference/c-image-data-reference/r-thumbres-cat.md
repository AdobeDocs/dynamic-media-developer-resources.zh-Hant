# ThumbRes{#thumbres}

縮圖解析度。 指定縮圖影像的物件解析度。

僅在`catalog::ThumbType=3`時使用。

## 屬性 {#section-ee5cae086a3d4875805fa8a08756a586}

實數值大於0。 必須與`catalog::Resolution`具有相同的單位。

## 預設 {#section-f0e453fc8a7c451db21961c084eecabd}

`attribute::ThumbRes` 如果欄位不存在、值為0或欄位空白，則會使用。
