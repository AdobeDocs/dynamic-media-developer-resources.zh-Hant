# Id {#id}

索引鍵值，通過該鍵值查找影像資料檔案中的記錄 [!DNL Platform Server]。

通常，如果SKU具有多個映像，則會有一個短且唯一的映像標識符，例如SKU編號，可能帶有某種類型的映像尾碼。 也可能是一個更複雜的字串，它看起來更像檔案路徑，支援使用Image Serving輕鬆重新擬合網站。

## 屬性 {#id-properties}

文本字串。 必要. 影像資料表的主索引鍵。 每個目錄：:Id值在表中必須唯一。

## 預設 {#id-default}

無。
