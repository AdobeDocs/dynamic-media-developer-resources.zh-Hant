# Id {#id}

Platform Server根據索引鍵值查找映像資料檔案中的記錄。

通常，如果SKU具有多個影像，則會是短且唯一的影像識別碼，例如SKU編號，可能會有某種影像尾碼。 也可能是更複雜的字串，看起來更像檔案路徑，以支援使用影像伺服輕鬆修復網站。

## 屬性 {#id-properties}

文字字串。 必要. 影像資料表的主索引鍵。 每個目錄：:Id值在表格中必須是唯一的。

## 預設 {#id-default}

無。
