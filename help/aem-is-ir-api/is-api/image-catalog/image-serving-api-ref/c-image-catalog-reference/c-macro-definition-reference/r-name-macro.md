# 名稱{#name}

宏名稱。

宏名稱可以包含ASCII字母、數字、「 — 」、「_」和「。」的任意組合。

## 屬性 {#section-3952a070c21646ab914cc76f289ed8ce}

文字字串。 必要. 每個宏：：名稱值在表中必須是唯一的。 宏名稱不區分大小寫。

## 預設 {#section-031d1931a7694da88c3f7d623eabcb78}

無。
