# 名稱{#name}

巨集名稱。

巨集名稱可包含ASCII字母、數字、&#39;-&#39;、&#39;_&#39;和&#39;.&#39;的任意組合。

## 屬性 {#section-3952a070c21646ab914cc76f289ed8ce}

文字字串。 必要. 每個巨集：：Name值在表格中必須是唯一的。 巨集名稱不區分大小寫。

## 預設 {#section-031d1931a7694da88c3f7d623eabcb78}

無。
