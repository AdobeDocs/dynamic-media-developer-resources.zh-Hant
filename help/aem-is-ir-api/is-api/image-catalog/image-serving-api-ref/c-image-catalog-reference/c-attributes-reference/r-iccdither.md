# IccDither{#iccdither}

顏色轉換抖動。 指定當未使用icc=進行明確選擇時，是否應使用抖動來改善顏色轉換的感知質量。

## 屬性 {#section-b7ba44d2d5de43f5a0841fe4cac29d35}

標幟. 將設定為0以禁用或設為1以啟用抖動。

## 預設 {#section-86c4230a16454464880f64d4ab5ad533}

如果未定義或為空，則從`default::IccDither`繼承。
