# Icc遞色{#iccdither}

色彩轉換遞色。 指定在沒有icc=明確選取時，是否應使用遞色來改善色彩轉換的感應品質。

## 屬性 {#section-b7ba44d2d5de43f5a0841fe4cac29d35}

標幟。 設為0可停用遞色，或設為1可啟用遞色。

## 預設 {#section-86c4230a16454464880f64d4ab5ad533}

如果未定義或空白，則繼承自`default::IccDither`。
