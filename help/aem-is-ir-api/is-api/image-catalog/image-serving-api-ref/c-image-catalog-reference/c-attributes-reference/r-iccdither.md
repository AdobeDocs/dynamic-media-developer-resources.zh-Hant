# IccDither{#iccdither}

色彩轉換混色。 指定當未使用icc=進行明確選取時，是否應使用混色來改善色彩轉換的知覺品質。

## 屬性 {#section-b7ba44d2d5de43f5a0841fe4cac29d35}

標幟. 設為0可停用或設為1可啟用混色。

## 預設 {#section-86c4230a16454464880f64d4ab5ad533}

繼承自 `default::IccDither` （如果未定義或為空）。
