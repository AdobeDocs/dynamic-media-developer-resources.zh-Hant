# RequestLock{#requestlock}

要求鎖定模式。 指定必須包含在要求中的要求鎖定型別。

## 屬性 {#section-ae89274d3b7940d0bf3d816f7c2f51fd}

標幟。 設為0會停用要求鎖定，設為1則會選取簡單的雜湊鎖定。

## 預設 {#section-fd5b5f0417bd444189b20494d179193e}

如果未定義或空白，則繼承自`default::RequestLock`。
