# 請求鎖{#requestlock}

請求鎖定模式。 指定請求中必須包含的請求鎖的類型。

## 屬性 {#section-ae89274d3b7940d0bf3d816f7c2f51fd}

標幟. 設定為0可禁用請求鎖定，設定為1可選擇簡單散列鎖。

## 預設 {#section-fd5b5f0417bd444189b20494d179193e}

繼承自 `default::RequestLock` 或為空。
