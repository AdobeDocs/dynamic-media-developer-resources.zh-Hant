# TotalStreamBitRate{#totalstreambitrate}

總資料流位元速率。 長數值，代表視訊資產的總資料流位元速率。

由`req=set`和`req=mbrSet`要求用來匯出位元速率資料。

僅適用於靜態內容目錄。

## 屬性 {#section-76731467b23240ac89b0767491c6020b}

整數值(bps)

## 預設 {#section-dba669ea75354671bbdb3b520e2f6911}

無。
