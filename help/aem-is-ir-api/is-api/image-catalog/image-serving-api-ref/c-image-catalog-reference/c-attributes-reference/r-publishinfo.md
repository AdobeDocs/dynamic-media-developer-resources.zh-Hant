# 發佈資訊{#publishinfo}

目錄注釋字串。 可用於標識目錄版本、發佈日期等的文本字串。

此欄位的內容將返回 `req=catalogprops`。 否則不使用此屬性。

## 屬性 {#section-73ff2a7c2e5b4c7a952453276f5bed23}

文本字串。

## 預設 {#section-82e5a885e142414f996c609efbb962b1}

繼承自 `default::PublishInfo` 或為空。
