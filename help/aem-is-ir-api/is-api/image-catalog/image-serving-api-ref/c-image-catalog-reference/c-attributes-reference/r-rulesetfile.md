# 規則集檔案{#rulesetfile}

規則集定義檔案路徑。 指定包含此映像目錄的規則集定義的檔案。

## 屬性 {#section-e8449b2225294518a5f9ddb698f68360}

文本字串值。 相對於目錄資料夾的絕對檔案路徑或路徑。

## 預設 {#section-20a9d535d64347abaf0233595b2ad089}

空，表示此映像目錄沒有專用請求預處理規則。
