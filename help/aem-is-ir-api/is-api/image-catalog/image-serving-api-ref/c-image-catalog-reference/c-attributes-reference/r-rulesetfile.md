# 規則集檔案{#rulesetfile}

規則集定義檔案路徑。 指定包含此影像目錄之規則集定義的檔案。

## 屬性 {#section-e8449b2225294518a5f9ddb698f68360}

文字字串值。 相對於目錄資料夾的絕對檔案路徑。

## 預設 {#section-20a9d535d64347abaf0233595b2ad089}

空白，表示此影像目錄沒有私人要求前置處理規則。
