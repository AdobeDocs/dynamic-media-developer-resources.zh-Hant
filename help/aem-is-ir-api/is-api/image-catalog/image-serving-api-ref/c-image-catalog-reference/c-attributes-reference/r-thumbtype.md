# ThumbType{#thumbtype}

預設縮略圖類型。 如果特定目錄記錄不包含有效的目錄：:ThumbType值，則為縮略圖類型提供預設值。

僅用於縮略圖請求( `req=tmb`)。

## 屬性 {#section-ae0babfe3c8e4c8ebe0124bc55051265}

枚舉。 允許的值為1、2和3, *`crop`*。 *`fit`*, *`texture`* 縮略圖類型。

## 預設 {#section-0237fcae4f304c5b876fceaa839b6b05}

繼承自 `default::ThumbType`或為空。
