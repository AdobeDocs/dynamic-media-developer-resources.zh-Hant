# 打印解析度{#printresolution}

預設打印解析度。 提供預設打印解析度，以備特定目錄記錄不包含有效目錄：:PrintResolution值時使用。

## 屬性 {#section-8f3cffb0f3f3406b81bc967366b241d6}

大於0的整數。 以每英吋點表示。

## 預設 {#section-b5d70865fe8d40229d8d91404fdf3244}

如果未定義或為空，則從`default::Resolution`繼承。
