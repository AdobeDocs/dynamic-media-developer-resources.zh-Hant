# BkgColor{#bkgcolor}

預設背景顏色。 RGB值，用於填滿不含實際影像資料之回覆影像的任何區域。

## 屬性 {#section-2768b2beee214676aa32f17e2a0e3343}

色彩.

## 預設 {#section-9c2c5794b82a4ef89a09551f08d03dbc}

繼承自 `default::BkgColor` （如果未定義或為空）。
