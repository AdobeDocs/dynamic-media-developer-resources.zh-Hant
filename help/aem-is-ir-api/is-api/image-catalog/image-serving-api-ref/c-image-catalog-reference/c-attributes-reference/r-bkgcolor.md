# BkgColor{#bkgcolor}

預設背景顏色。 RGB值，用來對不包含實際影像資料的回覆影像的任何區域進行填色。

## 屬性 {#section-2768b2beee214676aa32f17e2a0e3343}

顏色。

## 預設 {#section-9c2c5794b82a4ef89a09551f08d03dbc}

如果未定義或空白，則繼承自`default::BkgColor`。
