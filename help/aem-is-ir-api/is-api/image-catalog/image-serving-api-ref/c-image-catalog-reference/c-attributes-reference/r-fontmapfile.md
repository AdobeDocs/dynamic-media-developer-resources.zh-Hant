## 屬性 {#section-e88a0caab57542c598f57fa0e88dd254}

文本字串值。 相對於目錄資料夾的絕對檔案路徑或路徑。

## 預設 {#section-c57697856bc84d578e022ed3654fcd1c}

空，表示此影像目錄不包括專用字型。
