# FontMapFile{#fontmapfile}

字型對應檔案路徑。 指定包含此影像目錄的字型映射的檔案。

## 屬性 {#section-e88a0caab57542c598f57fa0e88dd254}

文字字串值。 相對於目錄資料夾的絕對檔案路徑或路徑。

## 預設 {#section-c57697856bc84d578e022ed3654fcd1c}

空白，表示此影像目錄不包含私用字型。
