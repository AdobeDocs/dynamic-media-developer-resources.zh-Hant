# ApplyEffectMask{#applyeffectmask}

可選擇在效果圖層上套用最終遮色片，以遮蔽效果與圖層之間的重疊區域。

有效值為0和1。 0表示沒有套用效果遮色片。 1表示套用效果遮色片。 預設值為1。
