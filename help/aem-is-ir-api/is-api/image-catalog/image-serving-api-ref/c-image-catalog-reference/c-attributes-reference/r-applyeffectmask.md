# 應用效果蒙版{#applyeffectmask}

可選地，在效果圖層上應用最終蒙版以遮蔽效果和圖層之間的重疊區域。

有效值為0和1。 0表示不應用效果蒙版。 1表示應用效果蒙版。 預設值為1。
