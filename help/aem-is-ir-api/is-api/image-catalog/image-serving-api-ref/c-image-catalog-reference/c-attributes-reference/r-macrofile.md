# 宏檔案{#macrofile}

宏定義檔案路徑。 指定包含此映像目錄宏定義的檔案。

## 屬性 {#section-e55829aecc3a412f9121a73d70f83a1a}

文字字串值。 相對於目錄資料夾的絕對檔案路徑或路徑。

## 預設 {#section-903de40cff734254aaf64c14258dc4a7}

空，表示此映像目錄沒有專用宏定義。
