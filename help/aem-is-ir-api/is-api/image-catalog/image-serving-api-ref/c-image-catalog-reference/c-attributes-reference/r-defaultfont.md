# 預設字型{#defaultfont}

預設字型名稱。 指定如果text= request未提供字型，應使用哪種字型。

## 屬性 {#section-b71618fec3c443a392d67eff480f0e5d}

文本字串。 如果指定，則必須是此影像目錄的字型映射或預設目錄的字型映射中的有效字型：:Name值。

## 預設 {#section-ed0199be32a24e31a47b345ce7d109f6}

繼承自 `default::DefaultFont` 或為空。
