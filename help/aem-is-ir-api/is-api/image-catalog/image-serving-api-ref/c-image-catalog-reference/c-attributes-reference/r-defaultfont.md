# 預設字型{#defaultfont}

預設字型名稱。 指定text=要求未提供字型時應使用的字型。

## 屬性 {#section-b71618fec3c443a392d67eff480f0e5d}

文字字串。 如果已指定，則必須是此影像目錄字型地圖或預設目錄字型地圖中的有效font：：Name值。

## 預設 {#section-ed0199be32a24e31a47b345ce7d109f6}

如果未定義或空白，則繼承自`default::DefaultFont`。
