# IccProfileMapFile{#iccprofilemapfile}

ICC設定檔對應檔案路徑。 指定包含此影像目錄之ICC設定檔對應的檔案。

## 屬性 {#section-962934fded9746c7953beb62616dd2a7}

文字字串值。 相對於目錄資料夾的絕對檔案路徑。

## 預設 {#section-c368cbef2a24435ea97cc7c83bbe2ce8}

空白，表示此影像目錄不包含私人ICC設定檔。
