# VideoBitRate{#videobitrate}

視訊位元速率。 允許指定與每個視頻記錄相關聯的比特率。

`req=set`和`req=mbrSet`請求用於導出位速率資料。

## 屬性 {#section-76731467b23240ac89b0767491c6020b}

整數值(bps)

## 預設 {#section-dba669ea75354671bbdb3b520e2f6911}

無。
