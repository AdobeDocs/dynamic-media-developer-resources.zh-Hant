# DefaultLocale{#defaultlocale}

預設地區ID。 未在請求中指定locale=時，用於本地化／翻譯的地區ID。

## 屬性 {#section-a101e9658d304e9caf032a99ecb80386}

文字字串。 如果`attribute::LocaleMap`和`attribute::LocaleStrMap`為空，或當`locale=`被指定為非空時，則忽略。
