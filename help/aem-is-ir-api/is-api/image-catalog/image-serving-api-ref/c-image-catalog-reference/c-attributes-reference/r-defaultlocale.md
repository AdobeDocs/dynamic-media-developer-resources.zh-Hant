# DefaultLocale{#defaultlocale}

預設地區ID。 未在請求中指定locale=時，用於本地化/翻譯目的的區域設定ID。

## 屬性 {#section-a101e9658d304e9caf032a99ecb80386}

文字字串。 如果`attribute::LocaleMap`和`attribute::LocaleStrMap`為空，或當`locale=`為指定且非空時，則忽略。
