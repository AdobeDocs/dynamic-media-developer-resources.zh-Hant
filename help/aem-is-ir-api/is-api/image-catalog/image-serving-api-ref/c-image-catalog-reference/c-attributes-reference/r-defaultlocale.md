# DefaultLocale{#defaultlocale}

預設地區ID。 要求中未指定locale=時，地區ID可供本地化/翻譯目的使用。

## 屬性 {#section-a101e9658d304e9caf032a99ecb80386}

文字字串。 如果`attribute::LocaleMap`和`attribute::LocaleStrMap`是空的，或指定`locale=`且不是空的，則會略過。
