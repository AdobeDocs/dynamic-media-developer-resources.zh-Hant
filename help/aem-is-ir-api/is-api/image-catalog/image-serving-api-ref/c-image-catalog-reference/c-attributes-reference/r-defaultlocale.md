# 預設區域設定{#defaultlocale}

預設區域設定ID。 在請求中未指定locale=時，用於本地化/翻譯目的的區域設定ID。

## 屬性 {#section-a101e9658d304e9caf032a99ecb80386}

文本字串。 如果忽略 `attribute::LocaleMap` 和 `attribute::LocaleStrMap` 為空，或 `locale=` 已指定且不為空。
