# 解析度{#resolution}

預設物件解析度。 提供預設物件解析度，以防特定目錄記錄未包含有效的目錄：:Resolution值。

## 屬性 {#section-1f65ea81c3324781af17d0321facbd19}

實數，大於0。 通常以每英吋像素表示，但也可以以其他單位表示，例如每米像素。

## 預設 {#section-2d1258d2b7ce4929ac327a5d41a108dd}

繼承自 `default::Resolution` （如果未定義或為空）。
