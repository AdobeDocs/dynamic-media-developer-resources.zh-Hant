# 解析度{#resolution}

預設對象解析。 在特定目錄記錄不包含有效目錄：:Resolution值時提供預設對象解析。

## 屬性 {#section-1f65ea81c3324781af17d0321facbd19}

實數，大於0。 通常以像素/英吋表示，但也可以以其他單位表示，如每米像素。

## 預設 {#section-2d1258d2b7ce4929ac327a5d41a108dd}

繼承自 `default::Resolution` 或為空。
