# VideoSize{#videosize}

視訊大小。 允許指定與每個視訊記錄相關聯的大小。

`req=set`要求用來匯出視訊大小資料。

## 屬性 {#section-93a2631a57e2423ea6b40f68c55062a1}

以逗號分隔的寬度和高度的整數值（例如，512,288）。

## 預設 {#section-baaee759ef344c54bbc014dc8944768a}

無。
