# VideoSize{#videosize}

視訊大小。 允許指定與每個視訊記錄相關的大小。

`req=set`請求用於匯出視訊大小資料。

## 屬性 {#section-93a2631a57e2423ea6b40f68c55062a1}

寬度和高度的逗號分隔整數值（例如512,288）。

## 預設 {#section-baaee759ef344c54bbc014dc8944768a}

無。
