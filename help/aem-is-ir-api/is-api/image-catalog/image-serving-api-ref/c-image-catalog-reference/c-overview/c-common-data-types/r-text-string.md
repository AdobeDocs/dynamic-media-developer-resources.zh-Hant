# 文字字串{#text-string}

文字字串資料型別。

除非字串包含任何`<CR>`、`<LF>`或`<TAB>`字元，否則字串分隔符號為選用。 單引號或雙引號均可作為分隔符號使用。 如果使用引號，則內嵌在字串中的任何引號都必須使用兩個連續的引號來逸出（例如This month&#39;s Special）。
