# 文字字串{#text-string}

文本字串資料類型。

字串分隔字元為選用字串，除非字串包含任何`<CR>`、`<LF>`或`<TAB>`字元。 單引號或雙引號可用作分隔字元。 如果使用雙引號，則字串中嵌入的任何此類引號都必須使用兩個連續引號來逸出(例如：本月的特別報告)。
