# 文字字串{#text-string}

文字字串資料型別。

字串分隔符號為選用，除非字串包含 `<CR>`， `<LF>`，或 `<TAB>` 個字元。 可使用單引號或雙引號做為分隔字元。 如果使用引號，則內嵌在字串中的任何引號都必須使用兩個連續引號來逸出（例如：本月的特殊引號）。
