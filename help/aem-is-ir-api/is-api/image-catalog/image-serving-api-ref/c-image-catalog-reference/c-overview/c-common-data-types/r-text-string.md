# 文本字串{#text-string}

文本字串資料類型。

字串分隔符是可選的，除非字串包含任何 `<CR>`。 `<LF>`或 `<TAB>` 字元。 單引號或雙引號可用作分隔符。 如果使用引號，則必須使用兩個連續的引號(如：本月的特別節目)。
