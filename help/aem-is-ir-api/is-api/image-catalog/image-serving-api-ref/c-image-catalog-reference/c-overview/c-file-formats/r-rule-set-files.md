# 規則集檔案{#rule-set-files}

規則集檔案是XML格式的文本檔案，必須遵守相應的標準和約定。

[!DNL RuleSet.xsd] 安裝在預設的目錄資料夾中，應用於在將規則集檔案提交到映像服務之前驗證它們。 應用嚴格分析，並應用不符合的規則集檔案 [!DNL RuleSet.xsd] 不要裝貨。
