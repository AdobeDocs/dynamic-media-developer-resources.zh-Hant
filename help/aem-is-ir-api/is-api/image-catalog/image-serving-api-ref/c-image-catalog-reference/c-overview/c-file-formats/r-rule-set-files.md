# 規則集檔案{#rule-set-files}

規則集檔案是XML格式的文字檔案，且必須符合適當的標準和慣例。

[!DNL RuleSet.xsd] 會安裝在預設目錄資料夾中，且應該用於在將規則集檔案提交至影像伺服之前對其進行驗證。 套用嚴格剖析且規則集檔案不符合 [!DNL RuleSet.xsd] 不要載入。
