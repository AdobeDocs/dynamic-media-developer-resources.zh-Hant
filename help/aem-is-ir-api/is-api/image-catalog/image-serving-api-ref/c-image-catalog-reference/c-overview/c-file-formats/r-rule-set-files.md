# 規則集檔案{#rule-set-files}

規則集檔案是XML格式的文字檔，必須符合適當的標準和慣例。

[!DNL RuleSet.xsd] 已安裝在預設目錄檔案夾中，且應用於驗證規則集檔案，然後再將它們送出至影像伺服。會套用嚴格的剖析，而不會載入不符合[!DNL RuleSet.xsd]的規則集檔案。
