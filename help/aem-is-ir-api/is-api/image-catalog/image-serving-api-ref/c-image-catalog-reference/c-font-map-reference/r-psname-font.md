# PsName{#psname}

Postscript字型面名。 Postscript字型面名。

目前，只有FXG轉譯器可以通過其Postscript名稱引用字型。

## 屬性 {#section-a560f8bfc69a4d88aeb0b53c1625153b}

文字字串。 選填。字型映射的次索引鍵。 相同的`font::PsName`值只能在同一字型映射檔案中出現一次。

## 預設 {#section-c5903d5340e740b69ec0150d2303a893}

無。
