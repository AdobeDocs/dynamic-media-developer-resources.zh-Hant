# Ps名稱{#psname}

Postscript字型名稱。 Postscript字型字名。

此時，只有FXG呈現器可以按其Postscript名稱引用字型。

## 屬性 {#section-a560f8bfc69a4d88aeb0b53c1625153b}

文本字串。 選擇性. 字型映射的輔助索引鍵。 相同 `font::PsName` 值只能在同一字型映射檔案中出現一次。

## 預設 {#section-c5903d5340e740b69ec0150d2303a893}

無。
