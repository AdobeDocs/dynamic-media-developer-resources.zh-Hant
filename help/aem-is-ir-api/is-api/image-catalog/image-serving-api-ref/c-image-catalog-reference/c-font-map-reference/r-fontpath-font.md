# FontPath{#fontpath}

字型檔案路徑。 主字型檔案的路徑和名稱，包括檔案尾碼。

## 屬性 {#section-ae2b7daf74a4419ebcec8f13aa8c1462}

文字字串。 必要. 必須有有效的映像伺服器檔案路徑，絕對或相對於`attribute::RootPath`。

## 預設 {#section-3fb42f362de84fdb8ba9fe4cc9628aa3}

無。
