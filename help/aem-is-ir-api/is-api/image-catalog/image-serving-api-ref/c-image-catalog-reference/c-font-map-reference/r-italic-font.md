# 斜體{#italic}

斜體標幟。

## 屬性 {#section-607e76490e87418496d1c6763018a14f}

標幟. 字型對應的選擇性第三索引鍵。

## 預設 {#section-0b88057d00ce4160929c23fbf138b309}

`0.`
