# 字型地圖參考{#font-map-reference}

如果影像目錄需要字型對映，則必須在屬性：：FontMapFile中指定字型對映檔案的相對或絕對路徑。

特定影像目錄的字型地圖專案會覆寫預設目錄的字型地圖專案。

「影像伺服」支援Adobe Type 1、OpenType和TrueType字型。

`textPs=`也支援CID、ATC和點陣字型。
