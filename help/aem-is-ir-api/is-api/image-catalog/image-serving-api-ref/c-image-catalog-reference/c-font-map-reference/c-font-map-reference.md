# 字型映射引用{#font-map-reference}

如果影像目錄需要字型映射，則必須在屬性：:FontMapFile中指定字型映射檔案的相對路徑或絕對路徑。

特定影像目錄的字型映射中的條目將覆蓋預設目錄的字型映射中的條目。

Image Service支援Adobe Type1、OpenType和TrueType字型。

`textPs=` 還支援CID、ATC和點陣圖字型。
