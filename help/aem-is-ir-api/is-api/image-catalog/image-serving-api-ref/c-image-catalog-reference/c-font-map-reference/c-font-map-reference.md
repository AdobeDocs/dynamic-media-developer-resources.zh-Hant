# 字型對應參考{#font-map-reference}

如果影像目錄需要字型圖，則必須在屬性：:FontMapFile中指定字型圖檔案的相對或絕對路徑。

特定影像目錄的字型映射中的項目將覆蓋預設目錄的字型映射中的項目。

Image Serving支援Adobe Type 1、OpenType和TrueType字型。

`textPs=` 也支援CID、ATC和點陣圖字型。
