# ICC配置式映射參考{#icc-profile-map-reference}

如果映像目錄需要ICC配置檔案映射，則必須在屬性：:IccProfileMapFile中指定配置檔案映射檔案的相對或絕對路徑。

特定影像目錄的ICC配置檔案映射中的條目覆蓋預設目錄的ICC配置檔案映射中的條目。

影像伺服支援符合ICC規格的色彩描述檔檔案。
