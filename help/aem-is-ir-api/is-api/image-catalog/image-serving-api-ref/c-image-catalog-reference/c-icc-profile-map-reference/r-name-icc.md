# 名稱{#name}

描述檔捷徑名稱。 顏色描述檔的捷徑名稱。

## 屬性 {#section-30fc50f71f4a4ea7b14b2524d1f2b698}

文字字串。 必要. ICC配置檔案映射的主索引鍵。 每 `icc::Name` 個值在表中必須是唯一的，且不能包含&#39;,&#39;字元。
