# 名稱{#name}

配置檔案快捷方式名稱。 顏色配置檔案的快捷方式名稱。

## 屬性 {#section-30fc50f71f4a4ea7b14b2524d1f2b698}

文字字串。 必要. ICC配置檔案映射的主索引鍵。 每個`icc::Name`值在表中必須是唯一的，並且不能包含「，」字元。

## 預設 {#section-47a56671d1e94497aa7ea83fe9590c05}

無。
