# ProfilePath{#profilepath}

設定檔檔案路徑。 ICC顏色配置檔案的路徑和名稱，包括檔案尾碼。

## 屬性 {#section-d342866853f24e168ba44215ef594e29}

文字字串。 必要. 必須是有效的影像伺服器檔案路徑，絕對路徑或相對於`attribute::RootPath`。

## 預設 {#section-838c08a4443648ffb2f5b9255fafa8dc}

無。
