# 座標{#coord}

畫素座標。 用於指定相對於影像或圖層矩形左上角的畫素位移形式的影像座標。 這些通常用於影像位移或裁切引數中。

座標 `0,0` 是指影像或矩形的左上角。 增加值會向右下方移動。
