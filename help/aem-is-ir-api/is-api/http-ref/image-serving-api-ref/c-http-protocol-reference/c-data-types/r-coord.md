# 坐標{#coord}

像素座標。 用於指定影像坐標，如影像偏移或裁切參數，其形式為相對於影像或圖層矩形的左上角的像素偏移。

 </tr>

0,0表示影像或矩形的左上角。 增加值會向右下方移動。
