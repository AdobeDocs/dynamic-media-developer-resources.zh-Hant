# 屬性{#properties}

屬性資料會回應下列`req=`類型：

* `catalogprops`
* `imageprops`
* `props`
* `userdata`

`userdata` 僅當內容遵循屬性格式時，才將響 `catalog::UserData` 應格式化為屬性。

* [文字(Java)屬性](r-text-java-properties.md)
* [JavaScript屬性](r-javascript-properties.md)
* [XML屬性](r-xml-properties.md)
* [JSONP屬性](r-json-properties.md)
