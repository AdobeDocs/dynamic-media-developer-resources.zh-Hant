# 屬性{#properties}

屬性資料會回應下列`req=`類型：

* `catalogprops`
* `imageprops`
* `props`
* `userdata`

`userdata` 只有當的內容遵循屬性格式時，響 `catalog::UserData` 應才會格式化為屬性。

* [文本(Java)屬性](r-text-java-properties.md)
* [JavaScript屬性](r-javascript-properties.md)
* [XML屬性](r-xml-properties.md)
* [JSONP屬性](r-json-properties.md)
