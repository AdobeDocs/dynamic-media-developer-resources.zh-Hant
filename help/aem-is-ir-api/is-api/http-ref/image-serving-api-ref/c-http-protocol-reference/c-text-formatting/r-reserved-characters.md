# 保留字元{#reserved-characters}

本節列出為特定目的而保留的字元。

RTF使用大括弧&#39; `{`&#39;和&#39; `}`&#39;作為組分隔符。 它們必須成對出現，並且可能是嵌套的。 要在文本字串中顯示大括弧，請使用「 `\{`&#39;和&#39; `\}`&quot; ,

>[!NOTE]
>
>您必須對RTF中使用的所有花括弧進行URL編碼。

反斜槓「\」用於引入RTF命令和關鍵字。 要顯示反斜線，請使用 `'\\'`。
