# 保留字元{#reserved-characters}

本節列出為特定目的而保留的字元。

RTF使用大括弧&#39; `{`&#39;和&#39; `}`&#39;作為群組分隔字元。 它們必須成對出現，而且可以巢狀。 若要在文字字串中顯示大括弧，請使用&#39; `\{`&#39;和&#39; `\}`&#39;的字元。

>[!NOTE]
>
>您必須對RTF中使用的所有大括弧進行URL編碼。

反斜線&#39;\&#39;用來引入RTF命令和關鍵字。 若要顯示反斜線，請使用 `'\\'`.
