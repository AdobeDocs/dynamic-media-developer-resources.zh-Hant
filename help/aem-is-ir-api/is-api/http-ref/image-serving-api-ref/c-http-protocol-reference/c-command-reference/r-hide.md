# 隱藏{#hide}

隱藏圖層。 允許隱藏圖層，而不需從請求字串中移除。

`hide=0|1`

## 屬性 {#section-09ff67290f064411baeb503a1aac2953}

層屬性。 為`layer=comp`指定時應用到層0。

## 預設 {#section-ec9f5130be1b435c84ba1bba54e38a48}

`hide=0` 可見圖層。
