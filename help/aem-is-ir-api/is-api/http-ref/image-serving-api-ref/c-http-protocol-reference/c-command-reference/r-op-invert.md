# op_invert{#op-invert}

反轉顏色。 反轉每個顏色元件以產生負面影像效果。

`op_invert=0|1`

## 屬性 {#section-2e7a7b340dd147db9bd68ff3bfb6f4c1}

圖層命令。 如果`layer=comp`，則套用至目前圖層或複合影像。 被效果圖層忽略。

## 預設 {#section-8040bc4e2a684c629a28b155aa0ba289}

`op_invert=0`，以避免顏色反轉。
