# op_invert{#op-invert}

反轉顏色。 反轉每個顏色分量以獲得負影像效果。

`op_invert=0|1`

## 屬性 {#section-2e7a7b340dd147db9bd68ff3bfb6f4c1}

圖層命令。 若`layer=comp`，則套用至目前層或複合影像。 被效果層忽略。

## 預設 {#section-8040bc4e2a684c629a28b155aa0ba289}

`op_invert=0`，無色彩反轉。
