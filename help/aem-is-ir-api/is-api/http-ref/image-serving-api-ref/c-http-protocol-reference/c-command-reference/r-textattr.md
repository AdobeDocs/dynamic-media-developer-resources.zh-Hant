## 屬性 {#section-114ca0b8585b403c873e2251478ad1d5}

文字圖層屬性。 被影像、純色和效果圖層忽略。 如果為`layer=comp`指定，則套用至`layer=0`。

## 預設 {#section-855230f5330b4afc8a933f00a1ed4612}

`textAttr=72,norm,fixedRes,wrap`
