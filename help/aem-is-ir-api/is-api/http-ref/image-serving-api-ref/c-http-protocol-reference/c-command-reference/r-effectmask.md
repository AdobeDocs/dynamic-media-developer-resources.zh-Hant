# effectMask{#effectmask}

覆寫請求的屬性值：:ApplyEffectMask。

effectMask是全域修飾元。 它會套用至請求中的所有效果圖層。

有效值為0和1。 0表示不套用效果遮色片。 1表示套用效果遮色片。 預設值為1。
