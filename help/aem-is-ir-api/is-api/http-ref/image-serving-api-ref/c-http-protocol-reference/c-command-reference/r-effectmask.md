# effectMask{#effectmask}

覆寫請求的屬性：:ApplyEffectMask的值。

effectMask是全域修飾詞。 它會套用至請求中的所有效果層。

有效值為0和1。 0表示不應用效果蒙版。 1表示應用效果蒙版。 預設為1。
