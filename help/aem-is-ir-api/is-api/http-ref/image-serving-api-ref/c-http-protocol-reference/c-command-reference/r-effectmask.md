# 效果掩碼{#effectmask}

覆蓋請求的屬性：:ApplyEffectMask的值。

effectMask是全局修飾符。 它應用於請求中的所有效果層。

有效值為0和1。 0表示不應用效果蒙版。 1表示應用效果蒙版。 預設值為1。
