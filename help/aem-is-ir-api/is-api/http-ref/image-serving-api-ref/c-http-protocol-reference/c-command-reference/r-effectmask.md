# effectMask{#effectmask}

覆寫要求的attribute：：ApplyEffectMask的值。

effectMask是全域修飾元。 它會套用至請求中的所有效果層。

有效值為0和1。 零(0)表示不要套用效果遮色片。 一(1)表示套用效果遮色片。 預設值為1。
