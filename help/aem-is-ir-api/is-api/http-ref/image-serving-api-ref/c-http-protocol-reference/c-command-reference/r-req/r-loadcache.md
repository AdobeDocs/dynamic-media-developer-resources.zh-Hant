# loadcache{#loadcache}

預先載入伺服器快取。 執行請求的方式與req=img類似，但伺服器不會傳回影像，而是傳回回回覆影像的長度(image.length)，格式化為MIME類型文字／純文字的文字資料。

`req=loadcache`

HTTP回應無法快取。

請求中的其他命令會如說明般套用。
