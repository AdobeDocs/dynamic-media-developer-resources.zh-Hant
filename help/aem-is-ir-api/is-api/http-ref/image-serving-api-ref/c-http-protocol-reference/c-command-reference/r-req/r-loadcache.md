# loadcache{#loadcache}

預載伺服器快取。 執行請求的方式與req=img類似，但伺服器不會傳回影像，而會傳回回覆影像的長度(image.length)，格式為MIME類型為text/plain的文字資料。

`req=loadcache`

HTTP回應無法快取。

請求中的其他命令適用於文檔。
