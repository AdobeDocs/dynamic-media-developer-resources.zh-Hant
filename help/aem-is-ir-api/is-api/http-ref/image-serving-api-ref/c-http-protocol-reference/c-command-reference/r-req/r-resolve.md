# 解決{#resolve}

除錯請求。 此調試命令解析並預處理請求、執行映像目錄查找、目錄：：修改項包含、宏和變數替代等，如req=img。

`req=resolve`

系統會傳回最終要求字串（而非結果影像），其MIME類型為`text/plain`。

HTTP回應無法快取。
