# 解決{#resolve}

除錯請求。 此debug命令可解析並預先處理請求、執行影像目錄查閱、目錄：：修飾詞包含、巨集和變數替代等，就像req=img。

`req=resolve`

會傳回最終請求字串，而非結果影像，MIME類型為`text/plain`。

HTTP回應無法快取。
