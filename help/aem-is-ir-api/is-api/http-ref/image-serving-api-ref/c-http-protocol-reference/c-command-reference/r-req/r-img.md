# img{#img}

影像（預設）。 要求標準影像資料。

`req=img`

回覆資料格式和回應MIME類型由`fmt=`決定。 `req=img` 為預設請求類型，不需要明確指定。HTTP回應可根據`catalog::Expiration`與TTL快取。

其他請求命令按文檔形式應用。
