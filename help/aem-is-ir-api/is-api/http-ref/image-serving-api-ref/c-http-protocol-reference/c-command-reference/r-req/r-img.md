# img{#img}

影像（預設）。 要求標準影像資料。

`req=img`

回覆資料格式和回應MIME型別由`fmt=`決定。 修飾元`req=img`是預設的要求型別，不需要明確指定。 HTTP回應可使用以`catalog::Expiration`為基礎的TTL進行快取。

其他要求命令會依記錄套用。
