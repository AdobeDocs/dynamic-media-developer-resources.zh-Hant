# img{#img}

影像（預設）。 要求標準影像資料。

`req=img`

回覆資料格式和回應MIME類型由`fmt=`決定。 `req=img` 是預設請求類型，不需要明確指定。HTTP響應基於`catalog::Expiration`可以與TTL進行快取。

其他請求命令會如檔案所述應用。
