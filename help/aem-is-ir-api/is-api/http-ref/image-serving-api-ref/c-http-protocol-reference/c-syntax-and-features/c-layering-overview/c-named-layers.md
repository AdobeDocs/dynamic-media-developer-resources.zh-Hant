# 已命名的圖層{#named-layers}

layer=指令允許為圖層指定名稱，並在稍後以名稱（而非圖層編號）參照相同的圖層。

這可讓範本定義已命名的圖層，然後可以從請求URL參照，而不需要使用變數。
