# 圖層順序{#layer-order}

圖層編號也會決定z順序。

圖層0 （背景圖層）是必要的；其他圖層編號不需要連續，而且會以圖層編號遞增的順序繪製在背景圖層的頂端。 具有最高圖層編號的圖層會呈現在頂端，且不會被其他圖層遮擋。
