# 圖層順序{#layer-order}

圖層編號也會決定z順序。

需要圖層0 （背景圖層）；其他圖層編號不需要連續，並且會以遞增圖層編號的順序在背景圖層上方繪製。 具有最高圖層編號的圖層會呈現在頂端，且永遠不會被其他圖層遮擋。
