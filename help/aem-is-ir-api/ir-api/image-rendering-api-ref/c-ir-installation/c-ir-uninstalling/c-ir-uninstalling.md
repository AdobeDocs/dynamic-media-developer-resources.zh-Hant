# 卸載影像渲染{#uninstalling-image-rendering}

如有必要，您可以解除安裝「影像演算」。

解除安裝可能會移除安裝資料夾中的部分使用者檔案。 在卸載之前備份要保留的所有檔案。

在卸載之前，請使用管理權限或根權限登錄到伺服器主機。
