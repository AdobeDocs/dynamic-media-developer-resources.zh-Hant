# 解除安裝影像演算{#uninstalling-image-rendering}

如果有必要，您可以解除安裝「影像演算」。

解除安裝可能會移除安裝資料夾中的部分使用者檔案。 在解除安裝之前，請備份您要保留的所有檔案。

解除安裝之前，請先以系統管理員或超級使用者許可權登入您的伺服器主機。
