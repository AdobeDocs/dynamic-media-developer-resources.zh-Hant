# 解除安裝影像呈現{#uninstalling-image-rendering}

如有必要，您可以解除安裝「影像轉譯」。

卸載可能會刪除安裝資料夾中的一些用戶文檔。 在卸載之前備份要保留的所有檔案。

在卸載之前，請使用管理權限或根權限登錄到伺服器主機。
