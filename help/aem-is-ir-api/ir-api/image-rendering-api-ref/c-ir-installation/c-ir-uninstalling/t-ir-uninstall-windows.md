# 在Windows上解除安裝{#uninstalling-on-windows}

請依照這些指示，在Windows系統上解除安裝「影像演算」。

1. 執行` *[!DNL install_folder]*\_uninst\uninstaller.exe`啟動解除安裝程式，或從[開始]功能表選取&#x200B;**[!UICONTROL 程式]** > **[!UICONTROL Scene7]** > **[!UICONTROL 影像演算]** > **[!UICONTROL 解除安裝]**。
