# 在Windows上卸載{#uninstalling-on-windows}

按照以下說明卸載Windows系統上的映像呈現。

1. 通過運行啟動卸載程式 ` *[!DNL install_folder]*\_uninst\uninstaller.exe`，或從「開始」菜單中選擇 **[!UICONTROL 程式]** > **[!UICONTROL Scene7]** > **[!UICONTROL 影像渲染]** > **[!UICONTROL 卸載]**。
