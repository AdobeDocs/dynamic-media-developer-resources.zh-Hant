# 在Windows上卸載{#uninstalling-on-windows}

請依照下列指示，在Windows系統上解除安裝「影像演算」。

1. 通過運行` *[!DNL install_folder]*\_uninst\uninstaller.exe`啟動卸載程式，或從「開始」菜單選擇「程式」>「Scene7」>「a4/」>「影像渲染」>「a6/」>「卸載」**[!UICONTROL 「卸載」&lt;a8/」。************]**
