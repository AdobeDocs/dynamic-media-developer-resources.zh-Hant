# 在Windows上卸載{#uninstalling-on-windows}

請依照下列指示，在Windows系統上解除安裝「影像演算」。

1. 執行解除安裝程 ` *[!DNL install_folder]*\_uninst\uninstaller.exe`式的方式，或從「開始」選單選 **[!UICONTROL 擇「程式]** > **[!UICONTROL Scene7]** > **[!UICONTROL Rendering Image]******>解除安裝程式」。
