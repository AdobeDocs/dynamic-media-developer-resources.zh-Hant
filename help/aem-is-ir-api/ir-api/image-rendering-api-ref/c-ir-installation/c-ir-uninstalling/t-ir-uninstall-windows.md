# 在Windows上解除安裝{#uninstalling-on-windows}

請依照下列指示，在Windows系統上解除安裝影像演算。

1. 執行以啟動解除安裝程式 ` *[!DNL install_folder]*\_uninst\uninstaller.exe`，或從「開始」功能表選取 **[!UICONTROL 計畫]** > **[!UICONTROL Scene7]** > **[!UICONTROL 影像演算]** > **[!UICONTROL 解除安裝]**.
