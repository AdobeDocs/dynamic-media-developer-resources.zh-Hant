# 授權{#licensing}

需要Dynamic Media影像演算授權金鑰，且必須安裝在已安裝影像演算的每個系統上。

如果沒有有效的授權，影像演算就不會執行。 安裝程式包括取得和安裝許可證金鑰的步驟。
