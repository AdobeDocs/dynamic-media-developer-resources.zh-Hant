# 許可{#licensing}

需要Dynamic Media影像呈現許可證密鑰，並且必須在安裝影像呈現的每個系統上安裝該密鑰。

沒有有效的許可證，影像呈現不會運行。 安裝過程包括獲取和安裝許可證密鑰的步驟。
