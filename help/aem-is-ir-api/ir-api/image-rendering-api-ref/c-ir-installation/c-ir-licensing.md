# 授權{#licensing}

需要有Dynamic Media影像渲染許可證密鑰，且必須在安裝影像渲染的每個系統上安裝該許可證密鑰。

沒有有效的授權，就無法執行影像演算。 安裝程式包括獲取和安裝許可證密鑰的步驟。
