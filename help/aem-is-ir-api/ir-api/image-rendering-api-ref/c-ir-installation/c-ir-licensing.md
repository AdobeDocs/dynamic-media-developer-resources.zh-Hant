# 授權{#licensing}

需要Dynamic Media Image Rendering授權金鑰，且必須安裝在安裝Image Rendering的每個系統上。

沒有有效的授權，就無法執行影像演算。 安裝程式包括獲取和安裝許可證密鑰的步驟。
