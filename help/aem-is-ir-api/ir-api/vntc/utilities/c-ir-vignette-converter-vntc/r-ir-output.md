# 輸出{#output}

vntc會產生文字資料，並傳送至stderr或記錄檔。

每個文本記錄的資料格式為一個`name=value`屬性。 字串值不加引號。 警告和錯誤消息始終會發送到`stderr`，即使指定了`-log`。

某些屬性可在同一輸出中多次發生。 序號（以0開頭）會附加至這些屬性的名稱，並以句點分隔。 這些屬性在下面以`. *`n`*`尾碼標識，尾碼為屬性名稱。
