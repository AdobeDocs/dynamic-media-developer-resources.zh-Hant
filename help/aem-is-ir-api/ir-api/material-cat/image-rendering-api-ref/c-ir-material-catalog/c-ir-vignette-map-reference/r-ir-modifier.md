# 修飾元{#modifier}

請求修飾詞字串。 無或多個由&#39;&amp;&#39;字元分隔的影像演算指令。 用來儲存範本和修飾元，這些範本和修飾元會在每次對此暈映對應項目提出要求時套用。

此欄位中的命令由引用此記錄的請求或模板中的相同命令覆蓋。

## 屬性 {#section-b7b01c307ebc41d48c34db77307dbfd1}

文字字串。 選填。

## 預設 {#section-31dd0ce4c0604fbaa7e7f1304626223b}

無。
