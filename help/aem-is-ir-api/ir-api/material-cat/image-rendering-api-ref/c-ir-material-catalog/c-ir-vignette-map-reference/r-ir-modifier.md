# 修飾元{#modifier}

請求修飾元字串。 沒有或多個以&#39;&amp;&#39;字元分隔的影像演算命令。 用來儲存每次要求此暈映對應專案時套用的範本和修飾元。

此欄位中的命令會由參考此記錄的請求或範本中的相同命令覆寫。

## 屬性 {#section-b7b01c307ebc41d48c34db77307dbfd1}

文字字串。 選擇性.

## 預設 {#section-31dd0ce4c0604fbaa7e7f1304626223b}

無。
