# 修飾元{#modifier}

請求修飾元字串。 沒有或多個以&#39;&amp;&#39;字元分隔的影像演算命令。 用來儲存每次要求此暈映對應專案時套用的範本和修飾元。

此欄位中的命令會由參考此記錄的請求或範本中的相同命令覆寫。
