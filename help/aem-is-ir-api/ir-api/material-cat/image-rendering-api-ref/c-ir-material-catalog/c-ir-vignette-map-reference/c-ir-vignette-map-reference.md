# 暈映地圖參考{#vignette-map-reference}

如果物料目錄需要暈映映射，則必須在屬性：:VignetteMapFile中指定暈映映射檔案的相對路徑或絕對路徑。

特定物料目錄的暈映映射檔案中的條目將覆蓋預設目錄的暈映映射檔案中的條目。
