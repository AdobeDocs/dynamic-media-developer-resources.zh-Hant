# Vignette地圖引用{#vignette-map-reference}

如果物料目錄需要維格特映射，則必須在屬性：:VignetteMapFile中指定維格特映射檔案的相對路徑或絕對路徑。

特定材料目錄的視圖映射檔案中的條目將覆蓋預設目錄的視圖映射檔案中的條目。
