# 使用者資料 *{#userdata}

使用者資料. 伺服器響應req=userdata，將此欄位的內容返回給客戶端。

新行可以用「??」逸出。 伺服器將替換「??」 在傳回此欄位的內容之前，使用`<cr><lf>`。

## 屬性 {#section-4e3904e047a84cc790ce77c56d3e0b21}

文字字串。 選填。

## 預設 {#section-a84b821918af41f5b84200f2edffacff}

無。
