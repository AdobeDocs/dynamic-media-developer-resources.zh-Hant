# 材料資料參考{#material-data-reference}

如果物料目錄需要物料資料，則必須在屬性：:CatalogFile中指定物料資料檔案的相對路徑或絕對路徑。

特定物料目錄的物料資料檔案中的條目覆蓋預設目錄的物料資料檔案中的條目。
