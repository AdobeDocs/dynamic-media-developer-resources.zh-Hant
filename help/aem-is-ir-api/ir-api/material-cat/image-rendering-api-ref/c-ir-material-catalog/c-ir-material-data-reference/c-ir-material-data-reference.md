# 材質資料參考{#material-data-reference}

如果材料目錄需要材料資料，則必須在attribute：：CatalogFile中指定材料資料檔案的相對或絕對路徑。

特定原物料型錄的原物料資料檔案中的專案，會修訂預設型錄的原物料資料檔案中的專案。
