# 重複{#repeat}

紋理重複模式。 指定紋理影像的拼貼方式以填充目標曲面。

## 屬性 {#section-cef4109cddf54ce095c3293d85bc412d}

列舉。 僅用於可重複的紋理。 忽略其他所有材料。
