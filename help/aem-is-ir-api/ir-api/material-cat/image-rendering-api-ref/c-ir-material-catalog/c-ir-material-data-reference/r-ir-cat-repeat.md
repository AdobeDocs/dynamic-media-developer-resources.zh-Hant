# 重複{#repeat}

紋理重複模式。 指定紋理影像平鋪以填充目標曲面的方式。

## 屬性 {#section-cef4109cddf54ce095c3293d85bc412d}

枚舉。 僅用於可重複紋理。 忽略所有其他材料。
