# Id{#id}

目錄記錄識別碼。 轉譯器伺服器用來尋找材料記錄的索引鍵值。 通常是簡短且唯一的資料識別碼，例如SKU編號。

## 屬性 {#section-9b20bf2545314620808b4210bce7afc9}

文字字串。 必要. 每個`catalog::Id`值在此素材目錄中必須是唯一的，且不得包含&#39;，&#39;字元。

## 預設 {#section-6a517046c43a40a7ba0af0212b5f31d2}

無。
