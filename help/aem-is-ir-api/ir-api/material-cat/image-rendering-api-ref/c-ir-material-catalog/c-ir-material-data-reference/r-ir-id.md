# Id{#id}

目錄記錄標識符。 呈現伺服器查看材料記錄的關鍵值。 通常是短且唯一的物料標識符，如SKU編號。

## 屬性 {#section-9b20bf2545314620808b4210bce7afc9}

文本字串。 必要. 每個 `catalog::Id` 值在此材料目錄中必須唯一，且不得包含「，」字元。

## 預設 {#section-6a517046c43a40a7ba0af0212b5f31d2}

無。
