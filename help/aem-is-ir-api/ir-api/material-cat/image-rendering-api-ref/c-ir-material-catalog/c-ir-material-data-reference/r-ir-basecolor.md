# 基本色彩{#basecolor}

主影像色彩。 減色顏色，用於紋理上色。 應該是紋理影像中的平均主色。

您也可以使用 `BaseColor`作為填色顏色，如果紋理材料沒有 `catalog::Color` 套用至不可紋理化的暈映物件。

## 屬性 {#section-d9aec235f28b402581cf2277f5fc6fbb}

色彩. 被純色和機櫃材料忽略。

## 預設 {#section-ca1813a6fcb24645b5835af0891b279d}

`0x808080` （中性灰色）。
