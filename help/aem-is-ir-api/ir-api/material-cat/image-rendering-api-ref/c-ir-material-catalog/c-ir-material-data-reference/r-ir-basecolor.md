# 基色{#basecolor}

主影像顏色。 減色，用於紋理著色。 應是紋理影像中的平均主色。

您還可以使用 `BaseColor`如果沒有 `catalog::Color` 應用於非紋理對象。

## 屬性 {#section-d9aec235f28b402581cf2277f5fc6fbb}

色彩. 被純色和機櫃材料忽略。

## 預設 {#section-ca1813a6fcb24645b5835af0891b279d}

`0x808080` （中性灰色）。
