# 輔助路徑{#auxpath}

資料檔案路徑。 與此映像關聯的非映像資料檔案的相對路徑和名稱。

伺服器將此值與屬性：:RootPath組合，以生成實際檔案路徑。 也可以是絕對檔案路徑。

用於為機櫃材料指定機櫃樣式檔案或為窗口覆蓋材料指定窗口覆蓋樣式檔案。 留空以備所有其它材料。

## 屬性 {#section-4268350054b7421da0ce0327f0731a52}

文本字串值。 如果指定，則它必須是有效的相對或絕對檔案路徑。 機櫃材料和窗蓋材料所需。 對於所有其他材料必須為空。

## 預設 {#section-287005c2d8e948fa958f69ba7b90b437}

無。
