# 伊盧姆{#illum}

照明圖選擇器。 允許在呈現此材料時顯式選擇照明圖。

## 屬性 {#section-162bcf562ca844ccba9e81e267508cca}

列舉。 設為–1，以根據catalog::Gloss的值自動選取照明地圖。

設定為0、1或2以選擇照度映射A、B或C。渲染器將選擇在暈映中可用的最接近的照度映射。

## 預設 {#section-ac386d31ef90423b8a367010a60bddc7}

-1（自動選擇）
