# 色彩{#color}

材料顏色。 實色材料的主色和彩色紋理材料的附加色。

## 屬性 {#section-073772b43b264b37bd96d1fb7404d10b}

色彩. 純色材料必需。 可選，用於可重複的紋理、機櫃和窗口覆蓋材料，以指定著色顏色。 如果不修改可重複紋理或機櫃材料的顏色，則此欄位必須保持為空。

## 預設 {#section-bf2c2ec642ed47da880ae2943b7d2c18}

無。
