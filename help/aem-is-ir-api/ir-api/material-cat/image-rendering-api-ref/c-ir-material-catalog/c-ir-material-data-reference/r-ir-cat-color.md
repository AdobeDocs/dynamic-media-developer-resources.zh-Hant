# 色彩{#color}

材料顏色。 固色材料的主色和著色紋理材料的添加色。

## 屬性 {#section-073772b43b264b37bd96d1fb7404d10b}

色彩. 實色材料所需。 可重複的紋理、機櫃和覆蓋材料的窗口可選，以指定顏色。 如果不修改可重複紋理或機櫃材料的顏色，則必須將此欄位留空。

## 預設 {#section-bf2c2ec642ed47da880ae2943b7d2c18}

無。
