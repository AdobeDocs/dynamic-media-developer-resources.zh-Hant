# 色彩{#color}

材質顏色。 純色材質的主要色彩，以及彩色紋理材質的主要色彩。

## 屬性 {#section-073772b43b264b37bd96d1fb7404d10b}

顏色。 純色材料的必要專案。 對於可重複材質、機櫃和視窗遮蓋材質，可選擇性指定上色顏色。 如果不修改可重複材質或封包材料的顏色，此欄位必須保留空白。

## 預設 {#section-bf2c2ec642ed47da880ae2943b7d2c18}

無。
