# 定義{#definition}

宏定義。 由一個或多個完整命令組成的Render Service協定片段，以「&amp;」字元分隔。

宏定義不能引用其他宏。 宏可以定義和使用自定義變數。

## 屬性 {#section-2584a79bd2a04ddeaa2c04fdac4835ae}

文本字串。 必要.

## 預設 {#section-4ab127aa653b4f76afa867c1acb1f31c}

無。
