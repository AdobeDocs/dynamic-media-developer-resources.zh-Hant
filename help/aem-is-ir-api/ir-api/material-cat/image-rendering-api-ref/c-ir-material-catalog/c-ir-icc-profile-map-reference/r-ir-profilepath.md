# ProfilePath{#profilepath}

配置檔案路徑。 ICC顏色配置檔案的路徑和名稱，包括檔案尾碼。

## 屬性 {#section-5b75d459083f49f3bbd89ce8458559f7}

文字字串。 必要. 必須是有效的映像伺服器檔案路徑，絕對路徑或相對於`attribute::RootPath`。

## 預設 {#section-67b79d62b3e24c62aff6844cc5e04251}

無。
