# ICC配置檔案映射參考{#icc-profile-map-reference}

如果影像目錄需要ICC配置檔案映射，則必須在屬性：:IccProfileMapFile中指定配置檔案映射檔案的相對路徑或絕對路徑。

特定物料目錄的ICC配置檔案映射中的條目將覆蓋預設目錄的ICC配置檔案映射中的條目。

影像呈現支援符合ICC規範的顏色配置檔案。
