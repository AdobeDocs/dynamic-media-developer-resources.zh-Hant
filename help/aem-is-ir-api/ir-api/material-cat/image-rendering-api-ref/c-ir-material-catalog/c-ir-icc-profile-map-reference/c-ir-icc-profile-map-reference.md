# ICC描述檔地圖參考{#icc-profile-map-reference}

如果映像目錄需要ICC配置檔案映射，則必須在屬性：:IccProfileMapFile中指定配置檔案映射檔案的相對或絕對路徑。

特定物料目錄的ICC配置檔案映射中的條目將覆蓋預設目錄的ICC配置檔案映射中的條目。

「影像渲染」支援符合ICC規範的顏色配置檔案。
