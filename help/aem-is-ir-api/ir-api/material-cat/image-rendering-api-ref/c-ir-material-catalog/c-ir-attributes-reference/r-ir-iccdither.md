# IccDither{#iccdither}

顏色轉換抖動。 指定在未進行顯式選擇時是否應使用抖動來改進顏色轉換的感知質量 `icc=`。

## 屬性 {#section-646fb48084734c66bf648360f3a5bfd1}

標幟. 設定為 `0` 禁用或 `1` 啟用抖動。

## 預設 {#section-c9066c361215404d847f4d2c8f1ea3a5}

繼承自 `default::IccDither` 或為空。
