# IccDither{#iccdither}

色彩轉換混色。 指定當未使用icc=進行明確選取時，是否應使用混色來改善色彩轉換的知覺品質。

## 屬性 {#section-646fb48084734c66bf648360f3a5bfd1}

標幟. 設為0可停用或設為1可啟用混色。

## 預設 {#section-c9066c361215404d847f4d2c8f1ea3a5}

如果未定義或為空，則繼承自`default::IccDither`。
