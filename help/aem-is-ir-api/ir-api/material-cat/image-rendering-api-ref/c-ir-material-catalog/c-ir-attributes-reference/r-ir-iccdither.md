# IccDither{#iccdither}

色彩轉換抖動。 指定當沒有明確選取時，是否應使用遞色來改善色彩轉換的感應品質 `icc=`.

## 屬性 {#section-646fb48084734c66bf648360f3a5bfd1}

標幟. 設定為 `0` 停用或 `1` 以啟用遞色。

## 預設 {#section-c9066c361215404d847f4d2c8f1ea3a5}

繼承自 `default::IccDither` 如果未定義或為空。
