# IccDither{#iccdither}

顏色轉換抖動。 指定當未使用icc=進行明確選擇時，是否應使用抖動來改善顏色轉換的感知質量。

## 屬性 {#section-646fb48084734c66bf648360f3a5bfd1}

標幟. 將設定為0以禁用或設為1以啟用抖動。

## 預設 {#section-c9066c361215404d847f4d2c8f1ea3a5}

如果未定義或為空，則從`default::IccDither`繼承。
