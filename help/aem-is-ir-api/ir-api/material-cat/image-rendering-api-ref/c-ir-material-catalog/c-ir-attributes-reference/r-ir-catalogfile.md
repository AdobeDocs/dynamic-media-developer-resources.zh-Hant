# CatalogFile{#catalogfile}

目錄資料檔案路徑。 指定包含此目錄材料資料的檔案。

## 屬性 {#section-81eaf83ed4d3405f8ba95671bd303584}

文字字串值。 相對於目錄資料夾的絕對檔案路徑或路徑。

## 預設 {#section-7a1f82b4188e41a3915a78744cada9fb}

空，表示此物料目錄不包含任何物料資料。
