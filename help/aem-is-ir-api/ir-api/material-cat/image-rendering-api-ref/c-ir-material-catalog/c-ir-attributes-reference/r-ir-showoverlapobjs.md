# ShowOverlapObjs{#showoverlapobjs}

顯示重疊對象。 指定重疊對象應最初顯示還是隱藏。

## 屬性 {#section-3e82e9b98d7a46f884d8b9798cec35c2}

標幟.

## 預設 {#section-e6f7e33a629b4592979bab6d14700e8f}

如果未定義或為空，則繼承自`default::ShowOverlapObjs`。
