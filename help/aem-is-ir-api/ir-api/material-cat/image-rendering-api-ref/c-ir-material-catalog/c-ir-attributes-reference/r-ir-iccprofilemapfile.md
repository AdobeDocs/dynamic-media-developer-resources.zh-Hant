# IccProfileMapFile{#iccprofilemapfile}

ICC設定檔對應檔案路徑。 指定包含此影像目錄之ICC設定檔對應的檔案。

## 屬性 {#section-33d55b55e192441bb30ca5eaaf07d12a}

文字字串值。 相對於目錄資料夾的絕對檔案路徑。

## 預設 {#section-191de4cf304544f0a4198ebd3b8ddfc2}

空白，表示此材料目錄不包含私人ICC設定檔。
