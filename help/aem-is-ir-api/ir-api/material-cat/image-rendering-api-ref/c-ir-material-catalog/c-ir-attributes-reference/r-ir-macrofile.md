# 宏檔案{#macrofile}

宏定義檔案路徑。 指定包含此材料目錄宏定義的檔案。

## 屬性 {#section-74ab431f12fb4fc9b1e222c752cfb4cb}

文字字串值。 相對於目錄資料夾的絕對檔案路徑或路徑。

## 預設 {#section-1ceb724a30444a5fbb870105b83c212c}

空，表示此材料目錄沒有專用宏定義。
