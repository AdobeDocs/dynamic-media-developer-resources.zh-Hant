# Tiff編碼{#tiffencoding}

TIFF編碼格式。 指定TIFF影像的壓縮格式(實際上為 `fmt=` )的正平方根。

設定為 `0` 無壓縮， `1` LZW `2` 用於縮減(ZIP), `3` JPEG壓縮。

## 屬性 {#section-469f5a1225464542866f5353edd92db3}

枚舉。

## 預設 {#section-a3c5152a9f464e4987ed7c05d35b1169}

繼承自 `default::TiffEncoding` 或為空。
