# VignetteMapFile{#vignettemapfile}

暈映映射資料檔案路徑。 指定包含此映像目錄暈映映射的檔案。

## 屬性 {#section-8f624cb8e49b44258a961fb7e464ce0a}

文字字串值。 相對於目錄資料夾的絕對檔案路徑或路徑。

## 預設 {#section-cc5c2e0ba54a4e5890a7330ee038fac8}

空，表示此材料目錄不包含暈映圖。
