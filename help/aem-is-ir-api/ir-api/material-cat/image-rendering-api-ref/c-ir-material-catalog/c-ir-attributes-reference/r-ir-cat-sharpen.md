# Sharpen{#sharpen}

## 屬性 {#section-faf57183efa0456fb37a7803a9486248}

標幟.

## 預設 {#section-e6f1f81ef5c949a488b2002581bda0d0}

如果未定義或為空，則繼承自`default::Sharpen`。
