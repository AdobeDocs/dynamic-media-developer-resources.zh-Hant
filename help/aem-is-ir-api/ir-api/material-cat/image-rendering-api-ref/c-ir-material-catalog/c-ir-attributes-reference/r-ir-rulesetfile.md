# RuleSetFile{#rulesetfile}

規則集定義檔案路徑。 指定包含此物料目錄的規則集定義的檔案。

## 屬性 {#section-e805898763824caba24abb6af1706f5c}

文字字串值。 相對於目錄資料夾的絕對檔案路徑或路徑。

## 預設 {#section-69b1f865aa8d476bb087dc862e903ab1}

空白，表示此材料目錄沒有私人請求預處理規則。
