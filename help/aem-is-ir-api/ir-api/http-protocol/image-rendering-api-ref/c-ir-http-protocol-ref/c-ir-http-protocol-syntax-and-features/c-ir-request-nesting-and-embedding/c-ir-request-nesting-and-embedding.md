# 請求巢狀內嵌*{#request-nesting-and-embedding}

影像演算可將影像伺服請求和HTTP請求內嵌至外來伺服器，以擷取用於可重複紋理和貼文的材質影像。

這些機構不支援機箱和窗口覆蓋材料。
