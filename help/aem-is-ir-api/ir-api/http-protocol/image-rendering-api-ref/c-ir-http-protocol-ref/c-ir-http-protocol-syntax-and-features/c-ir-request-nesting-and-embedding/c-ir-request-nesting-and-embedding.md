# 請求巢狀內嵌與內嵌{#request-nesting-and-embedding}

影像演算允許將影像伺服請求和HTTP請求內嵌到外部伺服器，以擷取用於可重複紋理和貼花的材料影像。

機櫃和視窗遮蓋材料不支援這些機制。
