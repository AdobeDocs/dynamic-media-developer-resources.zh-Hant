# 限制{#restrictions}

巢狀和內嵌會受到一些限制。

為獲得良好的伺服器效能，巢狀請求傳回的影像解析度應合理符合所套用材質之物件的紋理解析度。

外來影像會快取至本機。 只有在本機快取項目失效（根據過期的HTTP標題）後，才會偵測到此類影像的任何變更。
