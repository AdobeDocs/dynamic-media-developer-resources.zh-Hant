# 純色{#solid-colors}

此材料類別包括牆、木工和修剪的油漆、汽車，以及不需要紋理的任何其他應用。

例如，您可以使用實線顏色來更改不需要紋理更改的服裝項目上的顏色，或者設定汽車內飾的塑料修剪和地毯的顏色。 如果MSS中沒有`src=`值，則材料被視為實色。

實色可應用於所有可呈現的對象。
