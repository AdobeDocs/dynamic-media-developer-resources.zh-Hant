# 封包{#cabinets}

檔案櫃材料指定了檔案櫃樣式檔案（.vnc檔案副檔名）、包含檔案櫃的照片表示以及參數佈局定義和呈現檔案櫃前面所需的其他資訊的特殊資料檔案。

[!DNL vnc] 檔案可以包括可重複的木紋紋理，或者可以通過第二引數向外部提供紋理 `src=`。某些[!DNL vnc]檔案允許對選定的機櫃前面區域進行著色或紋理化（通常用於層壓機櫃樣式）。

機櫃材料只能應用於機櫃對象。
