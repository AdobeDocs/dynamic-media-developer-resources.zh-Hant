# 牆邊框{#wall-borders}

當在壁邊框MSS （隨sub=3..5引入）中指定材料時，它被視為壁邊框。

壁框線紋理影像可包含Alpha色版來定義框線的形狀。 壁邊界只能套用至壁物件。
