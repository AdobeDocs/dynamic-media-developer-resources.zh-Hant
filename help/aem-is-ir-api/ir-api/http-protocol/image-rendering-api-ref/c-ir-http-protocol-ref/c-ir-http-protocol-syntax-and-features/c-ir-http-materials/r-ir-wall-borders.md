# 牆邊框{#wall-borders}

當在牆邊框MSS （隨sub=3..5引入）中指定材料時，該材料會被視為牆邊框。

壁框線紋理影像可以包含Alpha色版來定義框線的形狀。 壁框線只能套用至壁物件。
