# 場景座標{#scene-coordinates}

場景座標空間用於指定可紋理化物件曲面上的大小和距離。

由於大部分的暈映都是描繪實體物件的真實世界場景，所以大部分的暈映都是使用英吋作為場景座標空間的單位來製作。 也可以使用其他單位，例如mm或cm。 影像演算不支援單位轉換。

下列指令接受場景座標空間中的值：

* [群組=](../../../../../../ir-api/http-protocol/image-rendering-api-ref/c-ir-http-protocol-ref/c-ir-http-protocol-command-reference/r-ir-grout.md#reference-73651cbbbc344adba2626ef950d3672a)
* [pos=](../../../../../../ir-api/http-protocol/image-rendering-api-ref/c-ir-http-protocol-ref/c-ir-http-protocol-command-reference/r-ir-pos.md#reference-22c10904a0ce4c8bb41c2c78104221b8)
* [大小=](../../../../../../ir-api/http-protocol/image-rendering-api-ref/c-ir-http-protocol-ref/c-ir-http-protocol-command-reference/r-ir-http-size.md#reference-1220d6fbcde4479aba91de7adacdc988)
