# 材質解析度{#material-resolution}

可重複紋理或其他材質的解析度值會定義場景座標空間中材質影像的大小。

紋理解析度值是以畫素/英吋（或暈映為其場景座標空間指定的任何單位）來指定。

材料解析度的指定方式為 `res=`.
