# 材料解析度{#material-resolution}

可重複紋理或其它材料的解析度值定義了場景坐標空間中材料影像的大小。

紋理解析度值以像素/英吋（或視頻為其場景坐標空間指定的任何單位）指定。

材料解析度指定為 `res=`。
