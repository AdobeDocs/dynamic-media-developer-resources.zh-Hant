# 光澤效果{#gloss-effects}

如果在暈映中啟用，則渲染器將改變照明算法以模擬材料光澤度的變化（主要通過控制對比度）。

此效果與反射渲染無關，通過模擬鏡面反射來補充它。 使用用於控制反射的相同`gloss=`、`glossmap=`和`type=`屬性來控制效果。
