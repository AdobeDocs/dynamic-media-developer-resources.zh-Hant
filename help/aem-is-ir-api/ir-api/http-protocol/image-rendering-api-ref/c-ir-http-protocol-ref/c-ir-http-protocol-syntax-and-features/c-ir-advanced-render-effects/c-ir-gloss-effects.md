# 光澤效果{#gloss-effects}

如果在暈映中啟用此功能，則轉換器將改變照明算法以模擬材料光澤度的變化（主要通過控制對比度）。

這種效果與反射演算無關，而且可模擬鏡面反射來彌補它。 使用用於控制反射的相同`gloss=`、`glossmap=`和`type=`屬性來控制效果。
