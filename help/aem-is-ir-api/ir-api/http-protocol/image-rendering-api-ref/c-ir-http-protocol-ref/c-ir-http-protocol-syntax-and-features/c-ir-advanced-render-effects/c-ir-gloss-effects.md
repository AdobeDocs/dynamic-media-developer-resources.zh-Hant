# 光澤效果{#gloss-effects}

如果在暈映中啟用此選項，則轉譯器會變更照明演演算法，以模擬材質光澤度的變化（主要是透過控制對比）。

此效果與反射彩現無關，而是透過模擬鏡面反射來補充。 效果是使用用於控制反射的相同`gloss=`、`glossmap=`和`type=`屬性來控制。
