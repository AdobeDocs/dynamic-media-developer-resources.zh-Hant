# 選取命令{#selection-commands}

這些命令用於選擇暈映組、對象、組或對象的子區域。

命令或材料（或兩者）在此選取命令之後指定，在下一個選取命令（或請求結尾）應用到選定項目之前指定。 選擇命令可分隔材料規範段(MSS)。
