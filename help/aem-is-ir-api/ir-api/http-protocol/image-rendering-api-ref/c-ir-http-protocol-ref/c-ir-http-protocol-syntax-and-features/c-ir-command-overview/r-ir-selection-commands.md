# 選取範圍指令{#selection-commands}

這些指令可用來選取暈映群組、物件、群組或物件的子區域。

這個指令（或材料，或兩者）是在這個選取指令之後並在下一個選取指令（或要求的結尾）套用至選取專案之前指定的。 選取指令可劃分材料規格區段(MSS)。
