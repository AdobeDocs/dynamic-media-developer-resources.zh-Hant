# illum{#illum}

照明地圖選取器。 指定此材質偏好呈現的照明對映。

`illum=-1|0|1|2`

如果目標暈映中沒有指定的照明對映，則會改用最近的可用對映。

`illum=-1`指定根據`gloss=`值自動選取照明地圖。

## 屬性 {#section-aace8466566e4cf1a0c5a6c0167245c9}

材質屬性。 如果暈映未定義多個照明對映，則忽略。

## 預設 {#section-c96ecfb232074e80b6a29076f5199403}

`illum=-1`
