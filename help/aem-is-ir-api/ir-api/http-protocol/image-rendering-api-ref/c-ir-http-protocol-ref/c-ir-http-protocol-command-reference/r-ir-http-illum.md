# 伊拉姆{#illum}

照明圖選擇器。 指定此材料要用來渲染的照明映射。

`illum=-1|0|1|2`

如果指定的照明映射在目標暈映中不可用，則會改用最接近的可用映射。

`illum=-1` 指定根據值自動選擇照明 `gloss=` 圖。

## 屬性 {#section-aace8466566e4cf1a0c5a6c0167245c9}

材料屬性。 如果暈映未定義多個照明映射，則忽略。

## 預設 {#section-c96ecfb232074e80b6a29076f5199403}

`illum=-1`
