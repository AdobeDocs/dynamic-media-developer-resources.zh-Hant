# 伊爾{#illum}

照明映射選擇器。 指定此材料希望用來渲染的照明映射。

`illum=-1|0|1|2`

如果指定的照明映射在目標視圖中不可用，則使用最近的可用映射。

`illum=-1` 指定根據 `gloss=` 值。

## 屬性 {#section-aace8466566e4cf1a0c5a6c0167245c9}

物料屬性。 如果視頻未定義多個照明映射，則忽略。

## 預設 {#section-c96ecfb232074e80b6a29076f5199403}

`illum=-1`
