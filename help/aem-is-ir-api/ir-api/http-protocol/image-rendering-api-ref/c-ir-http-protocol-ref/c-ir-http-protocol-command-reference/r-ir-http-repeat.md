隨機絎縫模式(14...18)可用於從不易重複的影像合成紋理；算法將根據原始影像建立完全隨機或部分隨機的紋理。

## 屬性 {#section-262bf540930d4890b678ea00cefe1909}

材料屬性。 被純色、傾斜和機櫃材料忽略。

## 預設 {#section-e5bbd7d9fbb74852849e605d20f550bb}

`catalog::Repeat`，則該材料基於目錄條目，否則( `0` 直重複)。
