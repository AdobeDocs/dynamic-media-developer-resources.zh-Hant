# 銳利化{#sharpen}

銳利化回覆影像。 演算後的影像根據wid=或hei=進行縮放後，可套用選擇性銳利化。

`sharpen=0|1`

 </tr> 
 <tr class="strow"> 
  <td class="stentry"> <p>1 </p></td>

 </tr> 
</table>
