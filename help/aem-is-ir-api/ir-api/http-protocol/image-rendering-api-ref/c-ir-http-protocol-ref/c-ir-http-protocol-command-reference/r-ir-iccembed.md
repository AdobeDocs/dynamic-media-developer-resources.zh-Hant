# iccEmbed{#iccembed}

內嵌ICC設定檔。 指定是否應將工作的ICC顏色配置檔案或使用icc=指定的配置檔案嵌入回復影像中。

`iccEmbed=0|1`

## 屬性 {#section-45c08073626a4cc7823d423dc35b4479}

可能發生在要求內的任何位置。

## 預設 {#section-68ed1f0c68dc46b58972804799c277b7}

`iccEmbed=0`，不會在輸出影像中嵌入ICC設定檔。
