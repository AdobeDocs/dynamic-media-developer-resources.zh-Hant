# 資料快取{#data-caching}

影像呈現會運用影像伺服快取功能來儲存回應影像資料、巢狀/內嵌請求的中間資料，以及從外部來源取得的來源影像資料（例如紋理）。

如需快取機制的說明，請參閱影像伺服檔案。
