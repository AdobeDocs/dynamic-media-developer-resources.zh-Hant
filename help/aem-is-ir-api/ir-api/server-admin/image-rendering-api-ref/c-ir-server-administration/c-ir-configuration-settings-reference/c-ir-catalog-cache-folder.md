# 目錄快取資料夾(ir.catalogCacheRoot){#catalog-cache-folder-ir-catalogcacheroot}

可以是絕對路徑或相對於[!DNL *[!DNL install_folder]*]的路徑。 預設值為[!DNL ./cache]，與[!DNL *[!DNL install_folder]*/cache]相對應。
