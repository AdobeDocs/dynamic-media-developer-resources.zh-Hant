# 目錄資料夾(ir.catalogRootPath){#catalog-folder-ir-catalogrootpath}

材料目錄資料夾的位置（必須位於所有catalog.ini檔案的位置）。

可以是絕對路徑或相對於[!DNL *[!DNL install_folder]*]的路徑。 預設值為[!DNL ./catalog]，與[!DNL *[!DNL install_folder]*/catalog]相對應。
