# 伺服器管理序言{#server-administration-preface}

本檔案說明伺服器管理問題，並說明Dynamic Media影像轉譯的組態設定。

**目標對象**

本檔案適用於需要安裝、設定和管理Dynamic Media影像演算的系統管理員和網站管理員。

**檔案慣例**

本檔案中描述的專案通常以下列專案型別為前置詞：
