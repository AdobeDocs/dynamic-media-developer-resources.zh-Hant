# 授權{#licensing}

若要使用影像伺服，需要Scene7 Image Serving授權金鑰，且必須安裝在要使用影像伺服的每個系統上。

同樣地，若要使用「影像演算」，則需要Scene7 Image Rendering授權金鑰。 影像演算可在沒有影像服務授權的情況下使用，反之亦然。

安裝程式包括獲取和安裝所需許可證密鑰的步驟。
