# 許可{#licensing}

要使用映像服務，需要Dynamic Media映像服務許可證密鑰，並且必須在要使用映像服務的每個系統上安裝該密鑰。

同樣，要使用「影像渲染」，需要使用「Dynamic Media影像渲染」許可證密鑰。 影像呈現可在沒有影像服務許可證的情況下使用，反之亦然。

安裝過程包括獲取和安裝必要的許可證密鑰的步驟。
