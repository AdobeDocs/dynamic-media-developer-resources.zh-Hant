# 授權{#licensing}

要使用映像服務，需要動態媒體映像服務許可證密鑰，並且必須在要使用映像服務的每個系統上安裝該密鑰。

同樣地，若要使用「影像演算」，則需要「動態媒體影像演算」授權金鑰。 影像演算可在沒有影像服務授權的情況下使用，反之亦然。

安裝程式包括獲取和安裝所需許可證密鑰的步驟。
