# 在Linux上啟動或停止{#starting-or-stopping-on-linux}

在Linux上啟動或停止映像服務有兩個選項。

* 啟動、停止和驗證映像服務狀態的指令碼位於[!DNL ImageServing/bin]資料夾中：

   `ImageServing.sh {start|stop|restart|status}`
* 系統管理員應熟悉下列替代方法：

   `/sbin/service ImageServing {start|stop|status}`
