# 安裝影像伺服 {#installation}

安裝影像伺服包含兩個主要伺服器元件：

* 影像伺服器。 獨立的原生程式碼可執行檔。
* [!DNL Platform Server]。 獨立式Java™ Web應用程式和應用程式伺服器。

「影像演算」是這兩個伺服器元件的一部分。

在後續的指示中，`install_root`會參照安裝「影像伺服」和「影像轉譯」的根資料夾。 且`server:port`是指設定為[!DNL Platform Server]之主要接聽連線埠的主機名稱或IP位址及連線埠號碼。
