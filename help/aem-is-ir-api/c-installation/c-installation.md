# 安裝{#installation}

您必須先安裝Image Serving，才能使用它。

影像伺服包含兩個主要元件：映像伺服器是可獨立執行的本地代碼，而平台伺服器是可獨立執行的Java Web應用程式和應用程式伺服器。 影像演算是這些伺服器元件的一部分。

在後續說明中， `install_root` 指安裝Image Serving和Image Rendering的根資料夾，並指 `server:port` Platform Server的主監聽埠配置的主機名或IP地址和埠號。
