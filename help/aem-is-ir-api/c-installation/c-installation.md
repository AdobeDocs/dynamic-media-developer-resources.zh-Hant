# 安裝{#installation}

您必須先安裝影像伺服器，才能使用它。

「影像伺服」包含兩個主要元件：映像伺服器是獨立的本機代碼執行檔，而平台伺服器是獨立的Java Web應用程式和應用程式伺服器。 影像呈現是這些伺服器元件的一部分。

在後續說明中，`install_root`指的是安裝了Image Serving和Image Rendering的根資料夾，而`server:port`指的是配置為Platform Server主偵聽埠的主機名或IP地址和埠號。
