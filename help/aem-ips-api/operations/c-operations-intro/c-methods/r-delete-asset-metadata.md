# deleteAssetMetadata{#deleteassetmetadata}

刪除資產的中繼資料值。 使用一系列中繼資料刪除，以設定批次中的值。

語法

## 授權的使用者型別 {#section-e913be43b684491daf02bc73211e4290}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>使用者必須擁有資產的讀取和刪除存取權。
