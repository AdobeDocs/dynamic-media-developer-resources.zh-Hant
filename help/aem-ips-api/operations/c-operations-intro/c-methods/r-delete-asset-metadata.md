# deleteAssetMetadata{#deleteassetmetadata}

刪除資產的中繼資料值。 使用中繼資料刪除的陣列，以在批次中設定值。

語法

## 授權用戶類型 {#section-e913be43b684491daf02bc73211e4290}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>使用者必須擁有資產的讀取和刪除存取權。
