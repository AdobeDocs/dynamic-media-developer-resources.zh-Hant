# forceRepublishAssets{#forcerepublishassets}

重設一或多個資產的發佈狀態，以強制在下一個發佈工作中重新發佈資產。

語法

## 授權用戶類型{#section-3d5a3e3afea748d69845de5c8c376448}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalContrib`
* `ImagePortalContribUser`
