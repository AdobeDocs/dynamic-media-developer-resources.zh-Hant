# addGroupMembership{#addgroupmembership}

將用戶添加到組陣列。

語法

## 授權用戶類型{#section-fe950150718a474d8df30d0f4453c022}

* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`

IPS API不會傳回此作業的回應。

此示例向具有` *`groupHandleArray`*`的公司添加組。 此範例僅使用一個群組。
