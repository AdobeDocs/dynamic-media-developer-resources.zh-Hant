# ImageSetMemberUpdate{#imagesetmemberupdate}

在此類型中， pageReset欄位對於RenderSet和目錄影像資產類型有意義：

* 對於`RenderSet`, `pageReset`指示新渲染視圖/色票組的開始。

* 對於目錄，`pageReset`表示新頁面檢視的開始。 每個頁面檢視通常有2個頁面影像，但您可以有更多或更少。
