# ExportJob{#exportjob}

作業類型，允許授權匯出先前上傳的檔案。

ExportJob不支援下列資產類型：

* 影像集
* 演算集
* 迴轉集
* 媒體集
* 多個位元速率設定
* 視訊集
* eCatalog
* 提案集

對於提供`fmt=convert`和`is_modifier`和`macro`的ExportJob請求，目標檔案採用`macro`提供的格式。 例如：

```
input_file = fileToExport.jpg
is_modifer = &fmt=png
macro=$test$ 
output_file = fileToExport.tiff
```
