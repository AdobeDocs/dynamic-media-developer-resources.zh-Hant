# ImageFormat{#imageformat}

非結構化的命名映像輸出設定集。

不是資產類型。

## 參數 {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| 名稱 | 類型 | 說明 |
|---|---|---|
| ` *`imageFormatHandle`*` | `xsd:string` | 影像格式控制代碼。 |
| ` *`名稱`*` | `xsd:string` | 影像格式名稱。 |
| ` *`urlModifier`*` | `xsd:string` | 代表影像格式的影像伺服器URL。 |
