# 不銳利化遮色片選項{#unsharpmaskoptions}

有助於改進優化金字塔TIF檔案的影像清晰度的設定。

`unsharpMaskOptions=[ *`數量、半徑、臨界值、單色`*]`

## 參數 {#section-c3f0d03136ba4422819cb463bd393885}

為`minOccurs=" *`n`*".`的`unsharpMaskOptions`選項指定值
