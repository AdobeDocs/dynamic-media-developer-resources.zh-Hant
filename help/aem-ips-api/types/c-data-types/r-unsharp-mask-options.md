# UnsharpMaskOptions{#unsharpmaskoptions}

有助於改善最佳化金字塔TIF檔案影像銳利度的設定。

`unsharpMaskOptions=[ *`數量、半徑、臨界值、單色`*]`

## 參數 {#section-c3f0d03136ba4422819cb463bd393885}

為選項指定 `unsharpMaskOptions` 值( `minOccurs=" *`n)`*".`
