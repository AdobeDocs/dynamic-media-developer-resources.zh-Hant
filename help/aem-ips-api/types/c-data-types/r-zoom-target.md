| 名稱 | 類型 | 說明 |
|---|---|---|
| ` *`zoomTargetHandle`*` | `xsd:string` | 縮放目標控制代碼。 |
| ` *`名稱`*` | `xsd:string` | 縮放目標名稱。 |
| ` *`xPosition`*` | `xsd:int` | 縮放目標的X位置。 |
| ` *`yPosition`*` | `xsd:int` | 縮放目標的Y位置。 |
| ` *`width`*` | `xsd:int` | 縮放目標寬度。 |
| ` *`height`*` | `xsd:int` | 縮放目標高度。 |
