# AssetContextStateUpdate{#assetcontextstateupdate}

為與資產相關聯的發佈上下文設定一組新的發佈狀態標幟。

**參數**

| 名稱 | 類型 | 說明 |
|---|---|---|
| ` *`assetHandle`*` | `xsd:string` | 處理您要更新的資產。 |
| ` *`contextStateUpdateArray`*` | `types:ContextStateUpdateArray` | 您要更新資產的發佈連絡人狀態陣列。 |
