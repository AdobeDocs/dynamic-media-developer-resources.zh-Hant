# [!DNL AssetCount]{#assetcount}

公司的資產型別和計數。

資產型別包括[!DNL ViewerSWf]、[!DNL PDF]、[!DNL Cabinet]等。

| 名稱 | 類型 | 說明 |
|---|---|---|
| assetType | `xsd:string` | 與計數相關聯的資產型別。 |
| 計數 | `sd:int` | 公司中`assetType`個資產的數量。 |
