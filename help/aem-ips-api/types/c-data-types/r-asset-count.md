# AssetCount{#assetcount}

公司的資產類型和計數。

資產類型包括[!DNL ViewerSWf]、[!DNL PDF]、[!DNL Cabinet]等。

| 名稱 | 類型 | 說明 |
|---|---|---|
| `*`assetType`*` | `xsd:string` | 與計數關聯的資產類型。 |
| `*`計數`*` | `xsd:int` | 公司中`assetType`資產的數量。 |
