# AssetCount{#assetcount}

公司的資產類型和計數。

資產類型 [!DNL ViewerSWf]包括 [!DNL PDF]、 [!DNL Cabinet]和其他。

| 名稱 | 類型 | 說明 |
|---|---|---|
| ` *`assetType`*` | `xsd:string` | 與計數關聯的資產類型。 |
| ` *`計數`*` | `xsd:int` | 公司 `assetType` 中的資產數。 |
