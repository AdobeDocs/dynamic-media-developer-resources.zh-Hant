**emailSetting選項**

`emailSetting`參數包含下列選項：

| 選項 | 退貨 |
|---|---|
| `All` | 指定收件者的所有工作通知（錯誤、警告、完成）。 |
| `Error` | 指定收件者的工作錯誤。 |
| `ErrorAndWarning` | 指定收件者的工作錯誤和警告。 |
| `JobCompletion` | 指定收件人的作業完成通知。 |
| `None` | 作業不會傳送任何作業通知給指定的收件者。 |
