**emailSetting選項**

`emailSetting`參數包含下列選項：

| 選項 | 傳回 |
|---|---|
| `All` | 指定收件人的所有作業通知（錯誤、警告、完成）。 |
| `Error` | 指定收件人的作業錯誤。 |
| `ErrorAndWarning` | 指定收件人的作業錯誤和警告。 |
| `JobCompletion` | 指定收件人的作業完成通知。 |
| `None` | 作業不會向指定的收件人發送任何作業通知。 |
