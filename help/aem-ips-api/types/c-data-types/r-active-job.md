# [!DNL ActiveJob]{#activejob}

在伺服器上運行的作業。 此外，它也是已排程作業的例項。

工作存在於三種狀態：

* 已排程執行。
* 當前正在運行。
* 已完成運行（並已將資訊寫入作業日誌）。

要返回作業類型，請指定作業類型值。 您可以傳回下列作業：

* `imageServingPublishJob`
* `imageRenderingPublishJob`
* `videoPublishJob`
* `serverDirectoryPublish`
* `JobUploadDirectoryJob`
* `uploadUrlsJob`
