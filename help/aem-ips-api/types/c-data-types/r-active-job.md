# ActiveJob{#activejob}

在伺服器上運行的作業。 此外，它也是排程工作的例項。

Job存在於3個狀態：

* 已排程執行。
* 目前正在執行。
* 已完成運行（並已將資訊寫入作業日誌）。

指定作業類型值以返回作業類型。 您可以傳回下列工作：

* `imageServingPublishJob`
* `imageRenderingPublishJob`
* `videoPublishJob`
* `serverDirectoryPublish`
* `JobUploadDirectoryJob`
* `uploadUrlsJob`
