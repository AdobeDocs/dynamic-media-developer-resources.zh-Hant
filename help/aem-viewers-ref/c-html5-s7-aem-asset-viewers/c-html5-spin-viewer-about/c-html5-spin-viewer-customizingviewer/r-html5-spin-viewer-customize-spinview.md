# 迴轉檢視{#spin-view}

主檢視包含迴轉影像。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

主要檢視器區域的&#x200B;**CSS屬性**

檢視區域的外觀是由下列CSS類別選取器所控制：

```
.s7spinviewer .s7spinview
```

```

 background-color: transparent; 
}
```
