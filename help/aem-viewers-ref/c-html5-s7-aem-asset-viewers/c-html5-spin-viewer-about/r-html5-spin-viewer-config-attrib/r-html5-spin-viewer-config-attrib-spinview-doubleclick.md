# SpinView.doubleclick{#spinview-doubleclick}

`[SpinView.|<containerId>_spinView.]doubleclick=none|zoom|reset|zoomReset`

<table id="table_E314540D347D47699C04EB80D20C0721"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> none|zoom|reset|zoomReset </span> </p> </td> 
   <td colname="col2"> <p> 配置按兩下／點選到回轉操作的映射。 設定為 <span class="codeph"> 無 </span> 會停用雙鍵／點選回轉。 如果設定縮放 <span class="codeph"> ，按一 </span> 下影像就會在一個回轉步驟中旋轉；CTRL+Click可旋轉一個回轉步驟。 設定為 <span class="codeph"> 重設 </span> 會導致在影像上按一下滑鼠，將回轉重設為初始回轉層級。 對於 <span class="codeph"> 縮放重 </span>設，如果目前的回轉系數達到或超過指定的限制，則會套用重設，否則會套用回轉。 </p> </td> 
  </tr> 
 </tbody> 
</table>

## 屬性 {#section-924163cb2f6542499f49894becef7fb5}

選填。

## 預設 {#section-7a2128fd488941948aff18421f533ccf}

`reset` 在桌上型電腦上；觸 `zoomReset` 控裝置。

## 範例 {#section-622348a84fbe4ff4b5dd7eb53b044d83}

`doubleclick=zoom`
