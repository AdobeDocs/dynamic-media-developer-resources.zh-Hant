# setHandlers{#sethandlers}

內嵌縮放檢視器的JavaScript API參考。

`setHandlers(handlers)`

指定零個或多個回呼處理常式。 對此方法的呼叫會完全覆寫先前為該檢視器執行個體指派的事件處理常式。 必須在`init()`之前呼叫。
