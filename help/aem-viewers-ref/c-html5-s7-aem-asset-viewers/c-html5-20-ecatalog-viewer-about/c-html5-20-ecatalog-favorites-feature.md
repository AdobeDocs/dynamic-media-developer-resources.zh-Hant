# 我的最愛功能{#favorites-feature}

一般使用者可以使用「我的最愛」功能，直接在e-catalog中標籤感興趣的產品，方法是將「心」圖示新增至e-catalog影像。 檢視器會將所有我的最愛儲存在使用者端系統上，並在同一使用者再次造訪電子目錄時再次顯示。

此外，檢視器可讓您檢閱以縮圖影像格點形式新增至e-catalog的所有我的最愛。

所有「我的最愛」工具都會分組在控制列中的一個下拉式清單下。
