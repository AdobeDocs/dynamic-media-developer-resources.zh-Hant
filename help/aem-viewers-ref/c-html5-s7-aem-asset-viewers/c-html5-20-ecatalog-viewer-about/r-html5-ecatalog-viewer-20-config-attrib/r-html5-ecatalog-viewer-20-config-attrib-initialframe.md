` initialFrame= *`frame`*`

   <td colname="col2"> <p> 指定要在檢視器載入時顯示的從零開始的展開索引。 索引符合橫向模式下的跨頁索引。 如果檢視器旋轉為縱向，檢視器會顯示跨頁所指向的最左側頁面 <span class="codeph"> frameIdx</span>. </p> </td>
