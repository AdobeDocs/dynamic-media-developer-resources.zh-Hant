# 下載{#download}

可以使用控制欄中的「下載」按鈕將電子目錄作為PDF檔案下載。

當出現以下情況時，「下載」按鈕在查看器用戶介面中自動可用：

* 客戶公司中存在實際的PDF檔案。
* 該名稱與傳遞給查看器的電子目錄資產的名稱匹配，並包括 `.pdf` 擴展。
* PDF檔案在Dynamic Media Classic發佈。
