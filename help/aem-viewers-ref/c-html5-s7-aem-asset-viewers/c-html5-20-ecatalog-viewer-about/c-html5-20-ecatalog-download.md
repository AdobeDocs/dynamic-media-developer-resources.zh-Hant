# 下載{#download}

您可使用控制列中的「下載」按鈕，將電子目錄下載為PDF檔案。

發生下列情況時，檢視器使用者介面中會自動提供「下載」按鈕：

* 客戶公司中有實際的PDF檔案。
* 此名稱與傳遞至檢視器的e-catalog資產名稱相符，並包含`.pdf`副檔名。
* PDF檔案會發佈在Dynamic Media經典。
