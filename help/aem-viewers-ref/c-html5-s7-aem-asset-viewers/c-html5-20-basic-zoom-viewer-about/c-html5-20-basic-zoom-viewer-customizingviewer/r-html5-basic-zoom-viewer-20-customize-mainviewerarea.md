# 主要檢視器區域{#main-viewer-area}

主要檢視區域是縮放影像所佔用的區域。 若未指定大小，通常會設定以符合可用的裝置熒幕。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

主要檢視器區域的&#x200B;**CSS屬性**

檢視區域的外觀是由下列CSS類別選取器所控制：

```
.s7basiczoomviewer
```

範例 — 設定具有白色背景( `#FFFFFF`)的檢視器，並將其大小設定為512 x 288畫素。

```
.s7basiczoomviewer{ 
 background-color: #FFFFFF; 
 width: 512px; 
 height: 288px;  
}
```
