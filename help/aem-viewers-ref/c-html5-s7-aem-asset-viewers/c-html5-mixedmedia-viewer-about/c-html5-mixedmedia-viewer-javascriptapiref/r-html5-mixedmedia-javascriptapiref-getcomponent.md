# getComponent{#getcomponent}

混合媒體檢視器的JavaScript API參考

`getComponent(componentId)`

傳回檢視器所使用之檢視器SDK元件的參考。 網頁可使用此方法來擴充或自訂現成檢視器的行為。 只有在`initComplete`檢視器回呼執行後才呼叫此方法，否則檢視器邏輯可能尚未建立元件。
