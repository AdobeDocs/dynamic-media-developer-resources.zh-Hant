# 回轉檢視{#spin-view}

當目前資產為回轉集時，主檢視由回轉影像組成。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**主查看器區域的CSS屬性**

查看區域的外觀由以下CSS類選擇器控制：

```
.s7mixedmediaviewer .s7spinview
```

  </tr> 
 </tbody> 
</table>

範例：讓回轉檢視透明。

```
.s7mixedmediaviewer .s7spinview { 
 background-color: transparent; 
}
```
