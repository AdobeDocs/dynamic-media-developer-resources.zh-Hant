# 導覽{#navigation}

視頻查看器的URL命令。

` navigation= *`file`*`

查看器支援通過托管WebVTT檔案進行視頻章節導航。 不支援提示定位運算子。

 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> <span class="varname"> 檔案</span> </span> </p> </td> 
   <td colname="col2"> <p> 指定WebVTT導航內容的URL或路徑。 映像服務應托管WebVTT檔案。 </p> </td> 
  </tr> 
 </tbody>
