# interactivedata{#interactivedata}

互動式視訊檢視器的URL命令。

`interactivedata= *`檔案`*`

互動式資料會將視訊內容中的特定時間區域，與稍後顯示在視訊旁邊互動式色票中的產品資料建立關聯。 它也會與視訊播放結束時顯示的行動號召面板相關聯。 此外，影片也會提供行動號召面板中所顯示的互動視訊標題。
