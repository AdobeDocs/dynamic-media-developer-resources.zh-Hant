# 導覽{#navigation}

互動式視訊檢視器的URL命令。

` navigation= *`file`*`

檢視器支援透過托管WebVTT檔案進行視訊章節導覽。 不支援提示定位操作器。

 <tbody>

   <td colname="col1"> <p> <span class="codeph"> <span class="varname"> 檔案</span> </span> </p> </td> 
   <td colname="col2"> <p> 指定WebVTT導航內容的URL或路徑。 影像伺服器應托管WebVTT檔案。 </p> </td>

選填。
