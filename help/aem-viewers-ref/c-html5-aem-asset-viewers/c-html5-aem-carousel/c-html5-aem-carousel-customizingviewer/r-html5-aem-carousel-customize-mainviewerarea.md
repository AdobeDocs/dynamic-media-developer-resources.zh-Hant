# 主觀看者區域{#main-viewer-area}

主要檢視區域是轉盤橫幅影像所佔用的區域。 當未指定大小時，通常會設定為適合可用的裝置畫面。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**主查看器區域的CSS屬性**

查看區域的外觀由以下CSS類選擇器控制：

```
.s7carouselviewer
```

範例：若要設定具有白色背景的檢視器(`#FFFFFF`)，並將其大小設為1174 x 500像素。

```
.s7carouselviewer { 
 background-color: #FFFFFF; 
 width: 1174px; 
 height: 500px;  
}
```
