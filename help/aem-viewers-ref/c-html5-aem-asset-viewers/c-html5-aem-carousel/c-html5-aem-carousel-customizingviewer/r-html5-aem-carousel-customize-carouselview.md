# 轉盤檢視{#carousel-view}

主檢視包含橫幅影像。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**主檢視器區域的CSS屬性**

檢視區域的外觀會使用下列CSS類別選擇器加以控制：

```
.s7carouselviewer .s7carouselview
```

示例——使主視圖透明。

```
.s7carouselviewer .s7carouselview { 
 background-color: transparent; 
}
```
