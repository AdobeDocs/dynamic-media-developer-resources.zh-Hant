# 鍵盤協助工具與導覽{#keyboard-accessibility-and-navigation}

檢視器使用者介面中公開的所有功能都可使用鍵盤。
一般使用者可以使用`Left`、`Right`、`UP`和`DOWN`按鍵來平移影像。
`Esc`按鍵可用來重設影像。
