# 鍵盤輔助功能和導航{#keyboard-accessibility-and-navigation}

查看器UI中公開的所有功能都可以使用鍵盤。
最終用戶可以使用 `Left`。 `Right`。 `UP` 和 `DOWN` 擊鍵來平移影像。
的 `Esc` 擊鍵可用於重置影像。
