# EmbedShare.embedsizes{#embedshare-embedsizes}

Video360 Viewer的設定屬性。

` [EmbedShare.|<containerId>_embedShare.]embedsizes= *`寬度`*, *`高度`*[,0|1][; *`寬度`*, *`高度`*[,0|1]]`

指定內嵌共用模式對話方塊中大小下拉式方塊的內嵌大小清單。
