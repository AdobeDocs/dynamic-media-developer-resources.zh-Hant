# setLocalizedTexts{#setlocalizedtexts}

Video360 Viewer的JavaScript API參考。

` setLocalizedTexts( *`localizationInfo`*)`

設定一或多個地區設定的本地化SYMBOL值。 必須在`init()`之前呼叫此引數。
