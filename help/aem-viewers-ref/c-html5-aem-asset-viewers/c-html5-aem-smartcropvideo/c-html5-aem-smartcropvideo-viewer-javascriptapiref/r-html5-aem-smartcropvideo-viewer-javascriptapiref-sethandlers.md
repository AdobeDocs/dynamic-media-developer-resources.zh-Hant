# setHandlers{#sethandlers}

用於Smart Crop Video Viewer的JavaScript API參考。

`setHandlers(handlers)`

指定零個或多個回調處理程式。 對此方法的調用將完全覆蓋以前為該查看器實例分配的事件處理程式。 必須在之前調用 `init()`。
