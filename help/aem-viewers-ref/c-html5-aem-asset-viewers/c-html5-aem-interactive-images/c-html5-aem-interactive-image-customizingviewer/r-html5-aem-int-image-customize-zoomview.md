# 縮放視圖{#zoom-view}

主視圖由靜態影像組成。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**主檢視器區域的CSS屬性**

檢視區域的外觀會使用下列CSS類別選擇器加以控制：

```
.s7interactiveimage .s7zoomview
```

```

 background-color: transparent; 
}
```
