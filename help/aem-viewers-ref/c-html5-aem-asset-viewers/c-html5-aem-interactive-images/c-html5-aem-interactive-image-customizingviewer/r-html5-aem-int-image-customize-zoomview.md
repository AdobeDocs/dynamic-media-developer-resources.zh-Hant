# 縮放檢視{#zoom-view}

主檢視由靜態影像組成。

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

主要檢視器區域的&#x200B;**CSS屬性**

檢視區域的外觀是由下列CSS類別選取器所控制：

```
.s7interactiveimage .s7zoomview
```

範例 — 讓主檢視透明。

```
.s7interactiveimage .s7zoomview { 
 background-color: transparent; 
}
```
