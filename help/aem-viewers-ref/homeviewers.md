# Dynamic Media觀眾參考指南{#dynamic-media-viewers-reference-guide}

自訂Dynamic Media檢視器，以便在桌上型電腦和觸控螢幕裝置上使用。 您可使用的媒體類型包括基本縮放、對話、縮放、對話搜尋、彈出、內嵌縮放、混合媒體、回轉、視訊、轉盤、維度、互動式視訊、互動式影像和視訊360檢視器。
